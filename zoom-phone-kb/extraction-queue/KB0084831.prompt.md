# Enabling or disabling Zoom Phone call summary templates

Account owners and admins can integrate [Zoom Phone call summaries](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0074868) with Custom AI Companion to meet business-specific formatting needs. This feature allows admins to create and manage [custom call summary templates](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0084830) and apply custom dictionaries to improve transcription accuracy across Zoom Phone communications, including call summaries, transcripts, voicemail transcriptions, and call recordings.

Learn more about [managing Zoom Phone call summary templates with AI companion](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0084830).

### Requirements for enabling or disabling Zoom Phone call summary templates

  * Either a Zoom Workplace license with Phone included, or a standalone Zoom Phone calling plan
  * Custom AI Companion add-on
  * [Call summary with AI Companion enabled](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0074867)
  * Account owner or admin privileges

## How to enable or disable Zoom Phone call summary templates

**Note** : This setting is only available with Zoom Workplace desktop app version 7.0.0 or later.

### Account

This setting allows account owners or admins to enable or disable **Call summary templates** for all users in the account.

  1. Sign in to the Zoom web portal as an account owner or admin.
  2. In the navigation menu, click **Account Management** then **Account Settings**.
  3. Click the **Zoom Phone** tab.
  4. Under **AI Companion** , click the **Call summary templates** toggle to enable or disable it.
  5. If a verification dialog appears, click **Enable** or **Disable** to verify the change.
  6. (Optional) To prevent all users in your account from changing this setting, click the lock icon , and then click **Lock** to confirm the setting.
  7. (Optional) Select a default call summary template.



### Group

This setting allows account owners or admins to enable or disable **Call summary templates** for a specific group of users. Changes apply only to users in the selected group.

  1. Sign in to the Zoom web portal as an admin with the privilege to edit groups.
  2. In the navigation menu, click **User Management** then **Groups**.
  3. Click the applicable group name from the list.
  4. Click the **Zoom Phone** tab.
  5. Under **AI Companion** , click the **Call summary templates** toggle to enable or disable it.
  6. If a verification dialog appears, click **Enable** or **Disable** to verify the change.  
**Note** : If the option is grayed out, it has been locked at the account level and needs to be changed at that level.
  7. (Optional) To prevent all users in the group from changing this setting, click the lock icon , and then click **Lock** to confirm the setting.
  8. (Optional) Select a default call summary template.



### Site

This setting allows account owners or admins to enable or disable **Call summary templates** for a specific site. Changes apply only to users on the selected site.

  1. Sign in to the Zoom web portal as an admin with the privilege to edit groups.
  2. In the navigation menu, click **Phone System Management** then **Company Info**.
  3. Click the applicable site name from the list.
  4. Click the **Policy** tab.
  5. Under **AI Companion** , click the **Call summary templates** toggle to enable or disable it.
  6. If a verification dialog appears, click **Enable** or **Disable** to verify the change.  
**Note** : If the option is grayed out, it has been locked at the account level and needs to be changed at that level.
  7. (Optional) To prevent all users in the site from changing this setting, click the lock icon , and then click **Lock** to confirm the setting.
  8. (Optional) Select a default call summary template.



### Phone user

This setting allows account owners or admins to enable or disable **Call summary templates** for a specific phone user. Changes apply only to the selected phone user.

  1. Sign in to the Zoom web portal as an admin with the privilege to edit groups.
  2. In the navigation menu, click **Phone System Management** then **Users & Rooms**.
  3. Click the **Users** tab.
  4. Click the applicable phone username from the list.
  5. Click the **Policy** tab.
  6. Under **AI Companion** , click the **Call summary templates** toggle to enable or disable it.
  7. If a verification dialog appears, click **Enable** or **Disable** to verify the change.  
**Note** : If the option is grayed out, it has been locked at the account level and needs to be changed at that level.
  8. (Optional) Select a default call summary template.



### Your personal account

This setting allows individual users to enable or disable **Call summary templates** for their personal account. You do not need admin permissions to change this setting.

  1. Sign in to the Zoom web portal.
  2. In the navigation menu, click **Settings**.
  3. Click the **Phone** tab.
  4. Click the **Settings** tab.
  5. Under **Others** , click the **Call summary templates** toggle to enable or disable it.
  6. If a verification dialog appears, click **Enable** or **Disable** to verify the change.  
**Note** : If the option is grayed out, it has been locked at either the group or account level. You cannot change it yourself and must contact your Zoom admin.
  7. (Optional) Select a default call summary template.
