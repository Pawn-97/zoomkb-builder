# Enabling calls from shared line groups by IP phone

Users can opt in or out of [shared line groups](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0068068) or [call queues](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0075507) by manually dialing via IP phone. This provides more flexibility for users who are members of multiple shared line groups. This feature gives users more control over their call participation _without_ [accessing the Zoom Workplace app](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0068068#h_1d82b228-8de9-4b6c-8afa-a3033fec22d3).

### Requirements for enabling or disabling calls from shared line groups by manually dialing via IP phone

  * Zoom Phone license
  * [Supported desk phone](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0061373)
  * Member of a shared line group  
**Note** : 
    * You can only belong to one shared line group. You can't belong to a call delegation setup and shared line group at the same time.
    * [View your shared line group name](https://support.zoom.com/hc/en/article?id=zm_kb&sysparm_article=KB0068068#h_1d82b228-8de9-4b6c-8afa-a3033fec22d3) to see if you're a member of a shared line group.

## How to enable or disable calls from shared line groups by manually dialing via IP phone

  1. Pick up the IP phone handset.
  2. Dial ***85** followed by the shared line group extension number.  
Once these digits are sent, if the conditions are matched, you will receive a confirmation prompt.



In the confirmation prompt, once the digits are sent, it will play a prompt that says "You are now opted out" or "You are now opted in".
