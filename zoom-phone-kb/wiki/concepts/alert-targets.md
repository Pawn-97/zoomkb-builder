# Alert Targets

## Summary

Determines the scope of monitoring for each alert rule. The three target types (Account, Specify Device up to 5, Site) are conditionally available based on the selected rule. This conditional availability means the target dropdown must dynamically filter options based on the rule selection, a design consideration for form ordering.

## Key points

- Account: monitors all devices in the account (available for Devices go offline/online rules only)
- Specify Device (up to 5): monitors individually selected desk phones by user or extension (available for Devices go offline/online rules only)
- Site: monitors all devices at a site (available for Offline Devices Rate rule only)
