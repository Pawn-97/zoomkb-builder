# Auto-Receptionist Caller ID Configuration

## Summary

Auto-receptionist numbers can serve as outbound caller IDs, but improper configuration may cause intermittent call failures. The caller ID must be explicitly set in the user's profile, and the feature requires correct policy and calling plan alignment. Designers should consider warning indicators when an auto-receptionist number may cause issues.

## Key points

- Auto-receptionist numbers can be used as outbound caller ID
- Incorrect configuration can lead to intermittent call failures
- Requires explicit setup in user profile under Outbound Caller ID settings
- Post-configuration testing is recommended to verify calls succeed
