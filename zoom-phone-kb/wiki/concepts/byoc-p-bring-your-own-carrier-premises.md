# BYOC-P (Bring Your Own Carrier - Premises)

## Summary

The recommended configuration for normal-mode call handling when an SBC is integrated with Zoom. BYOC-P routes calls through standard Zoom cloud infrastructure during normal operation and automatically fails over to ZPLS when cloud connectivity is lost. This hybrid approach gives designers a model where normal-mode UX and survivability-mode UX are distinct paths.

## Key points

- Recommended for normal-mode call handling with SBC
- Processes calls through Zoom cloud infrastructure during normal operation
- Automatic failover to ZPLS when cloud connectivity lost
