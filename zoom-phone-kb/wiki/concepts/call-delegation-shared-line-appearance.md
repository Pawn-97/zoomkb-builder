# Call Delegation (Shared Line Appearance)

## Summary

A feature that allows phone users to assign assistants or colleagues to handle calls on their behalf, also known as shared line appearance. From a UX perspective, delegation creates a dual-role relationship where delegates can answer, place, and manage calls for the delegator, requiring clear visual indicators to distinguish between acting on one's own behalf versus on behalf of another user.

## Key points

- Enables phone users to designate others to make and receive calls on their behalf
- Functions at multiple hierarchy levels: Account, Group, Site, Phone User, and Common Area
- Requires enabling by an admin before users can set up delegation relationships
