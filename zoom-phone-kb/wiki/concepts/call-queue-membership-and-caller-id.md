# Call Queue Membership and Caller ID

## Summary

A user's ability to select a call queue number as their outbound caller ID depends directly on whether they are a member of that call queue. If a user needs to display a call queue number, an admin must first add them as a member. This membership-based availability model creates a dependency that the UI should make transparent.

## Key points

- Only call-queue member numbers appear in the outbound caller ID dropdown
- Admins must add users to a call queue before the number becomes selectable
- Users who leave a call queue lose access to that number as caller ID
- Multiple call queue numbers can be added as caller ID options per user
