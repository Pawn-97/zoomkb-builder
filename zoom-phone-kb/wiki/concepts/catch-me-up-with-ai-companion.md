# Catch Me Up with AI Companion

## Summary

A feature that lets users request an AI-generated summary of an ongoing push-to-talk conversation from a selected point in time. The UI presents multiple time-based catch-up options, and the AI returns a structured summary that helps users quickly rejoin the conversation context.

## Key points

- Accessible via an AI Companion icon in the top-right of the call screen
- Presents multiple time-based options for how far back to summarize
- Returns structured output: time period, quick recap, next steps, full summary
- Useful for users joining a push-to-talk channel mid-conversation
