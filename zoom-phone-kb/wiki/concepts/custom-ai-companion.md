# Custom AI Companion

## Summary

An add-on to Zoom's AI Companion that enables custom call summary templates and dictionary management. It is required for the call summary template feature to work. Designers should treat this as a feature-gating license -- without it, template functionality is unavailable, so the UI should gracefully handle the unlicensed state.

## Key points

- Required add-on for call summary templates
- Group members with a Custom AI Companion license can use templates assigned to them
- Powers the AI prompt-based template generation and custom dictionary features
