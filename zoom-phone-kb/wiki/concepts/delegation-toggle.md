# Delegation Toggle

## Summary

A simple enable/disable toggle for the delegation feature, consistently used across all hierarchy levels (Account, Group, Site, Phone User, Common Area). The toggle is the primary interaction for controlling delegation availability, and its consistent placement and behavior across levels reduces admin learning curve.

## Key points

- Same toggle interaction across all five hierarchy levels
- Requires verification dialog confirmation on change
- Grayed-out state indicates the setting is locked at a higher hierarchy level
- Optional lock icon allows enforcement at current level to prevent lower-level override
