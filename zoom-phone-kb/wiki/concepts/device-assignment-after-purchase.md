# Device Assignment After Purchase

## Summary

After purchasing hardware, customers must assign the device to a specific user to complete setup. If forgotten, the system sends reminders via email and the Billing Portal. This post-purchase assignment step is critical to the hardware activation workflow and should be prominently surfaced after checkout.

## Key points

- Device assignment to a user is required to complete hardware setup
- Email and Billing Portal reminders are sent if assignment is not completed
- Assignment likely happens in a separate interface from the purchase flow
