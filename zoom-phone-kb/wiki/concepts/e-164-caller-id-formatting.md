# E.164 Caller ID Formatting

## Summary

Zoom Phone uses E.164 international formatting for all caller ID display, which includes country code prefixes by default and cannot be disabled. Designers must account for the fact that display behavior also depends on the recipient's device, carrier, and phone settings, meaning the caller ID format is not fully controllable.

## Key points

- Country code prefixes are always included and cannot be removed
- Japanese +81 prefix is specifically called out as immutable
- Display varies by recipient device, carrier, and phone settings
- No per-user option to suppress international formatting
