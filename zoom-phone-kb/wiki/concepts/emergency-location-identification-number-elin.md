# Emergency Location Identification Number (ELIN)

## Summary

A dedicated BYOC number required for emergency services support during survivability mode. The ELIN is configured at the Site level and must remain unassigned to any user or Zoom Phone entity. Designers should ensure the admin interface clearly distinguishes the ELIN from regular BYOC forwarding numbers and prevents accidental assignment.

## Key points

- Required for emergency services during survivability mode
- Configured per Site, must not be assigned to any user or entity
- Separate from call forwarding BYOC numbers
