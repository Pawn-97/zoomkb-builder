# eSIM Management

## Summary

The centralized admin interface within the Zoom web portal for managing Zoom Phone Mobile eSIMs. Admins can create eSIM packages for users, issue and manage eSIMs, view roaming details, and control activation states. It provides a single pane of glass for mobile connectivity administration.

## Key points

- Accessible under Phone System Management > eSIM Management
- Manages eSIM creation, issuance, deactivation, and reissuance
- Separate from User Management—license assignment requires two-step process
- Automatically creates a mobile device endpoint for the user in call handling settings
- Displays users, packages, numbers, status, roaming details, and available actions
- Search bar for finding specific users
- Ellipsis menu per user row for actions like View, Issue new eSIM
