# eSIM Package

## Summary

The combination of a Zoom Phone user, a Zoom Phone Mobile package (license type), and a phone number that constitutes a fully provisioned eSIM service. Creating a package automatically provisions the subscriber, generates eSIM details, and triggers a welcome email with a QR code.

## Key points

- Three components: user, package (license tier), and phone number
- Auto-provisions subscriber and eSIM on creation
- Triggers welcome email with QR code for user activation
- Can be disabled, enabled, or reissued by admin
