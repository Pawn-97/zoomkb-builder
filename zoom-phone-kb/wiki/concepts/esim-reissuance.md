# eSIM Reissuance

## Summary

The process of issuing a new eSIM for a user, which deactivates the previous eSIM and generates a new QR code welcome email. Used when a user loses their device or an eSIM needs to be regenerated.

## Key points

- Previous eSIM is automatically deactivated
- New QR code sent via email
- Available from both list view (Action column) and user detail page
- Requires admin confirmation in a pop-up modal
