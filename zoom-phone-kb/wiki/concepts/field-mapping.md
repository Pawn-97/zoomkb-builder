# Field Mapping

## Summary

The mechanism by which Zoom Phone call data is passed to third-party applications in a Zap. Available fields include Caller/Callee Name, Phone Number, Extension details, Call ID, Direction, and DateTime. These fields can be inserted into email subjects, CRM records, spreadsheet rows, etc.

## Key points

- 16 available Zoom Phone fields covering caller, callee, call metadata, and forwarding info
- Fields include both identity (names, numbers) and context (direction, timestamp, call ID)
- Design implication: field naming and descriptions should make downstream mapping intuitive
