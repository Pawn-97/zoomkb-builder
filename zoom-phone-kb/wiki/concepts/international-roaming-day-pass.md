# International Roaming Day Pass

## Summary

A time-bounded roaming add-on that admins can enable per user. Roaming is disabled by default and operates on a day-pass model: the pass is only activated when the user actually starts roaming, not at purchase time. Admins can set start and end dates per roaming plan.

## Key points

- Disabled by default for all new eSIM packages
- Day pass activates on actual roaming, not on purchase
- Date range specified by admin (start and end dates)
- Can be added, edited, or deleted per user
- Helps limit unintended roaming charges
