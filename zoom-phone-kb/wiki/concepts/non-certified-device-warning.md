# Non-Certified Device Warning

## Summary

A persistent red warning message displayed in the web portal when a desk phone model is not on Zoom's certified list. The warning informs admins the device may not work with Zoom Phone features and cannot be dismissed, making it a high-visibility alert that demands attention.

## Key points

- Persistent red warning message — does not auto-dismiss or fade
- Alerts admins that the device is not supported for Zoom Phone features
- Appears for any device model not in the certified list
