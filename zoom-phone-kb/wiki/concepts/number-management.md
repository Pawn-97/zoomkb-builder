# Number Management

## Summary

The centralized interface within the Zoom web portal where admins manage phone numbers, including viewing, assigning, and transferring them between accounts. It supports both Zoom Phone and Zoom Contact Center product scopes.

## Key points

- Accessible via navigation menu under Number Management > Phone Numbers
- Displays ellipsis menus per number for individual actions and checkbox selection for bulk operations
- Move-to-account flow uses a guided pop-up wizard with dropdowns for target account, product, subscription, division, and SIP group
