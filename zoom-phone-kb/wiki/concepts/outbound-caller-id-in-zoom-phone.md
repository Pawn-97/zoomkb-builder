# Outbound Caller ID in Zoom Phone

## Summary

A configurable setting that determines which phone number appears on a recipient's caller ID display when a Zoom Phone user makes an outbound call. Users can select from numbers assigned to their account, the company main number, or call queues they belong to. The UX must clearly present available options within the 100-number dropdown limit and explain why certain numbers may not appear.

## Key points

- Caller ID can be set to user's own number, company main number, or call queue numbers
- Selection is restricted to numbers the user is explicitly associated with
- Dropdown has a maximum display limit of 100 phone numbers
- Auto-receptionist numbers can be used as outbound caller ID with proper configuration
