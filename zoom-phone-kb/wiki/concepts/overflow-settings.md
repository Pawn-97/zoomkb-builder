# Overflow Settings

## Summary

Configuration that defines what happens to calls that are not answered within a call queue. Common overflow destinations include voicemail, call forwarding to another number, or routing to a different call queue. Essential for ensuring no call is dropped.

## Key points

- Triggered when no member in the queue answers
- In group rotating, triggered after all groups have been tried
- Can route to voicemail, external forwarding, or another call queue
