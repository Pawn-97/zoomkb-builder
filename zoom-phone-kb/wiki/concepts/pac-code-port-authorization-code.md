# PAC Code (Port Authorization Code)

## Summary

A code obtained from a current UK business mobile service provider that authorizes number porting to Zoom Phone Mobile. The code is valid for 30 days, creating a time-sensitive constraint on the porting workflow. UX must surface the expiry deadline and prompt users to complete the porting process before the code becomes invalid.

## Key points

- Obtained from the current business mobile service provider
- Valid for 30 days from issuance
- Required alongside the current supplier name for porting
- Only applicable when porting existing numbers, not for new number provisioning
