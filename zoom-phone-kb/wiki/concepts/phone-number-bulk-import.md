# Phone Number Bulk Import

## Summary

CSV-based workflow to populate phone numbers for the entire LTI course roster at once. Admins download a template, fill in numbers, and upload it back. This pattern is commonly used for batch operations where manual entry would be impractical.

## Key points

- Downloadable CSV template provides the correct format
- Upload replaces manual per-contact number entry
- Import dialog guides the user through the three-step process
