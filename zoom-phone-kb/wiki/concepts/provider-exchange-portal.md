# Provider Exchange Portal

## Summary

An external portal used for phone number provisioning that shows country availability. When countries appear as unavailable, this could indicate missing verification prerequisites in the Zoom web portal rather than a provider-level restriction.

## Key points

- Shows country availability for number provisioning
- May display countries as unavailable if verification prerequisites are incomplete
- Works in conjunction with the Zoom web portal for address verification
