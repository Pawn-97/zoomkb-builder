# Shared Access

## Summary

A permission model on the Zoom App Marketplace that allows an app to act on behalf of all users in an account, not just the authenticating user. For Zapier, this enables Zaps to trigger for any user's phone activity, making it suitable for team-wide or organization-wide automation.

## Key points

- Requires User with Shared Access or Admin permission on Zoom App Marketplace
- Selected via a checkbox during OAuth consent flow
- Enables app-wide trigger scope vs. single-user scope
- Design implication: checkbox must clearly explain its scope impact
