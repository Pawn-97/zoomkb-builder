# Shared Line Group Opt-in/out via IP Phone Star Codes

## Summary

A mechanism for users to toggle their participation in shared line groups or call queues by dialing a star code (*85 + extension) from a physical desk phone. This provides call-center-style agent state control without requiring the Zoom Workplace desktop or mobile app. The audible confirmation tone tells the user whether they are now opted in or opted out.

## Key points

- Dial *85 followed by the shared line group extension number from an IP phone handset
- Works for both shared line groups and call queues
- Audible voice prompt confirms 'You are now opted out' or 'You are now opted in'
- Designed for users who do not or cannot use the Zoom Workplace app to manage call participation
