# Source-Destination Number Mapping

## Summary

The pre-configured forwarding logic that defines which Zoom native numbers (sources) redirect to which BYOC numbers (destinations) during an outage. Multiple source numbers can map to a single destination number. Sources can come from Users, Common Area Phones, Share Line Groups, Call Queues, and Auto Receptionists, making this a flexible but complex mapping interface.

## Key points

- Many-to-one mapping: multiple source numbers can forward to single destination
- Sources can be from Users, Common Area Phones, Share Line Groups, Call Queues, Auto Receptionists
- Default forwarding status is disabled; must be individually enabled
- Main company number cannot be used as a source number
