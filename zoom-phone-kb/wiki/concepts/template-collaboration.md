# Template Collaboration

## Summary

A feature that controls who can edit call summary templates. Two modes: 'Everyone' (all users can edit) and 'Restricted' (only the creator and explicitly added collaborators can edit). This pattern balances collaborative iteration with governance control, important for organizations where template content may have compliance implications.

## Key points

- Two collaboration modes: Everyone (open edit access) and Restricted (creator + named collaborators only)
- Collaborators appear in a managed list
- Accessible via Settings icon on the template page
