# Trigger Events

## Summary

Zoom Phone events that can initiate a Zap workflow. Available triggers include: Phone Call Connected, Phone Call Ended, Phone Call History Completed, Phone Call Recording Completed, and Phone Call Voicemail Received. Each represents a distinct moment in the call lifecycle.

## Key points

- Five trigger events covering the full call lifecycle
- Connected triggers at call start; Ended/Completed at call conclusion
- Recording Completed and Voicemail Received are media-specific triggers
- Design implication: clear event taxonomy helps users select the right trigger
