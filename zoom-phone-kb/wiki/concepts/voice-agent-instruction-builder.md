# Voice Agent Instruction Builder

## Summary

The interface within Zoom Virtual Agent where voice agent behavior is defined through skills, triggers, and instructions. The SMS tool is inserted into instructions using a search or forward-slash shortcut, with dynamic variables for sender number, recipient number, and message content. Designers should ensure the variable insertion experience is intuitive for non-technical admins building conversation flows.

## Key points

- Skills section defines trigger name and description
- Instruction section is where tools like SMS are inserted
- Forward-slash (/) shortcut or Insert button to add tools/variables
- Content field supports dynamic variables for message personalization
