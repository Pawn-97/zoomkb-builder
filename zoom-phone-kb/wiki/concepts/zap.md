# Zap

## Summary

The core workflow unit in Zapier: a combination of a trigger event from one app and one or more actions in other apps. For Zoom Phone, a Zap might trigger on a completed call and create a CRM contact or send an email with call details.

## Key points

- Composed of a trigger (Zoom Phone event) and one or more actions (third-party app)
- Created in Zapier's workflow builder interface
- Requires Zoom account to be connected/authenticated within Zapier
