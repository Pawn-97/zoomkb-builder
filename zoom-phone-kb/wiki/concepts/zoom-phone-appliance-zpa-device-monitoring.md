# Zoom Phone Appliance (ZPA) Device Monitoring

## Summary

An alerting system that allows admins to monitor the online/offline status of Zoom Phone Appliance devices and receive notifications when thresholds are crossed. Three alert rule types cover individual device status changes and site-level offline percentages. This is a proactive monitoring feature designed to surface hardware issues before end users report them.

## Key points

- Monitors ZPA devices for online/offline status changes
- Three alert rules: devices go offline, devices go online, and offline devices rate
- Alerts can be targeted at individual devices, specific sites, or the entire account
