# Zoom Phone for Zapier Integration

## Summary

A no-code integration that connects Zoom Phone with Zapier to automate phone-related workflows. Users can create Zaps that trigger on Zoom Phone events (call ended, voicemail received, recording completed, SMS) and perform actions in thousands of third-party apps like CRMs, email, and messaging platforms. It eliminates manual data entry and enables real-time call-driven automations.

## Key points

- Trigger-based: events like call ended, voicemail received, recording completed, SMS received
- Action capability: send SMS to up to 10 recipients from Zapier
- Field mapping: Zoom Phone data (caller name, number, direction, timestamp) mapped to third-party app fields
- OAuth authentication with shared access option for app-wide triggers
