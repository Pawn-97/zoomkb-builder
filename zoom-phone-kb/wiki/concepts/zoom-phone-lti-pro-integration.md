# Zoom Phone + LTI Pro Integration

## Summary

Embeds Zoom Phone calling capability directly into LTI Pro within an LMS course. Instructors and admins can make calls, send SMS, and view recordings without leaving their course environment. The integration relies on the Zoom client for call routing, meaning users must be signed into both the LMS and the Zoom desktop client.

## Key points

- Click-to-dial from within LTI Pro for automatic phone number detection
- Floating phone window powered by Zoom Phone Smart Embed
- Call logs, voicemails, and SMS accessible within the LTI interface
- Uses Zoom client to route calls—separate sign-in required
