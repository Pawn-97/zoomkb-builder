# Zoom Phone Mobile

## Summary

A service that extends Zoom Phone to employees' mobile devices using eSIM technology, making the smartphone a native Zoom Phone endpoint. Users make and receive business calls through their phone's native dialer while retaining enterprise features (presence, voicemail, call recording) from the Zoom Phone platform. Calls can route over cellular voice networks for carrier-grade reliability.

## Key points

- eSIM-based integration with mobile network providers
- Native dialer for calls; Zoom Workplace app for advanced features
- Business and personal communication separation on one device
- Centralized admin control via eSIM Management in the web portal
- Per-user license, assigned via eSIM Management (not User Management)
- Two types: Primary line (50GB data, 20GB hotspot) and Secondary line (2GB data, no hotspot)
- License type must match device SIM configuration or data connectivity is auto-deactivated
- Must assign Zoom Phone Basic license in User Management before assigning Mobile license in eSIM Management
- Must be assigned by an account owner or admin before the user can activate eSIM
- Only assignable to Zoom Phone users (not usable by other products)
- Requires a Zoom Phone license or qualifying Zoom Workplace plan
- UK-specific mobile eSIM service for business customers
- Only business mobile numbers are supported, not personal numbers
- Temporary numbers available before verification completes
- Number porting uses PAC codes from existing providers
