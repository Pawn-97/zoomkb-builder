# Zoom Phone SMS Feature

## Summary

SMS messaging capability within Zoom Phone that, when combined with Zoom Virtual Agent, enables automated text messaging during voice calls. The feature requires an approved messaging campaign for the phone number and an allow list of users who can view incoming SMS replies. The dual-configuration nature (ZVA side + ZP side) means designers must consider two separate setup interfaces working in concert.

## Key points

- Requires both Zoom Virtual Agent and Zoom Phone licenses
- Phone number must have an Approved Messaging Campaign
- Allow List controls which users can view incoming customer SMS replies
- SMS toggle is under Auto Receptionist Policy > General section
