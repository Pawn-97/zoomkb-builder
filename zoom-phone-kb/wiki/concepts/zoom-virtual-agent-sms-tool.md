# Zoom Virtual Agent SMS Tool

## Summary

An integration that allows Zoom Virtual Agent voice agents to send SMS messages during live calls, enabling agents to deliver rich content (URLs, survey links, dynamic text) that would be difficult to convey over voice alone. Designers must consider how the SMS interaction co-exists with the ongoing voice call and how recipients experience the transition between modalities.

## Key points

- Embeds SMS capabilities within voice agent interactions during live calls
- Messages can contain dynamic text, URLs, or survey links
- Configured as a 'tool' within the voice agent's instruction builder
- Uses forward-slash (/) keyboard shortcut for variable and tool insertion
