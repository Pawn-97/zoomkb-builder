# 30-Day Return Window with Unopened Box Requirement

## Summary

Hardware returns and refunds are available within 30 days of purchase, but only if the box remains unopened. Returns are handled through UC Direct's RMA form and return policy. This strict condition should be clearly communicated before purchase and reinforced in post-purchase communications.

## Key points

- 30-day return window from date of purchase
- Box must be unopened to qualify for return/refund
- RMA form and return process managed by UC Direct, not Zoom
