# Account and Subscription Prerequisites

## Summary

Call Forwarding Local Survivability requires Business/Education/Enterprise account tier, Zoom Node and Zoom Phone subscriptions, admin privileges, and a deployed ZPLS module. These are consistent across all ZPLS features, indicating a shared eligibility gate in the admin experience.

## Key points

- Business, Education, or Enterprise account
- Zoom Node and Zoom Phone subscriptions required
- ZPLS service module must be deployed
