# Account-Only Scope with No Subordinate Controls

## Summary

The conference join/leave tone can only be toggled at the account level — there are no group, site, or individual user overrides. Designers should note that this is a simpler configuration model compared to other Zoom Phone settings that support hierarchical locking, and it means all users share the same experience.

## Key points

- Only configurable at account level — no group, site, or user options
- No lock mechanism needed since there are no lower-level settings to lock
- Enabled by default for all users
- Does not affect shared line appearance conference-join settings
