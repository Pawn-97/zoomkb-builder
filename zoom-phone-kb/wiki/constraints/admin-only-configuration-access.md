# Admin-Only Configuration Access

## Summary

The call data transfer setting is only accessible to account owners and admins with edit privileges. This means that support staff and contact center supervisors who might benefit from understanding this data flow cannot see the setting. Designers should consider whether an info-only view for non-admin roles would improve transparency.

## Key points

- Setting is invisible to non-admin users
- Only account owners and admins with edit privileges can modify
