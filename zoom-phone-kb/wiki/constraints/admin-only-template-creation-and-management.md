# Admin-Only Template Creation and Management

## Summary

Only account owners and admins can create, edit, and manage templates. End users are limited to using templates assigned to them. This means all template customization must go through an admin, which could create bottlenecks in organizations that need frequent template adjustments.

## Key points

- Only account owners and admins can create and manage templates
- End users can only use assigned templates
- Collaborators can edit specific templates when granted access
