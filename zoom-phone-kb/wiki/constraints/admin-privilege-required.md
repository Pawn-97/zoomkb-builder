# Admin Privilege Required

## Summary

Only account owners or admins with appropriate privileges can enable or disable delegation. Different hierarchy levels require different privilege scopes (edit account settings, edit groups, edit sites), which means the same admin may have access at some levels but not others.

## Key points

- Account owner or admin privileges required at all levels
- Group level requires privilege to edit groups
- Site level requires privilege to edit sites
- Account level requires privilege to edit account settings
