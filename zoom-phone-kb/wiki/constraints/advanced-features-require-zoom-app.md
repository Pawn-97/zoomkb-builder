# Advanced Features Require Zoom App

## Summary

While basic calling works through the native dialer, advanced Zoom Phone features (voicemail, presence, call transfer, recording) are only accessible through the Zoom Workplace mobile app. This split between native dialer and Zoom app creates a dual-interface experience that designers must account for in onboarding and feature discovery.

## Key points

- Native dialer: basic call making and receiving only
- Zoom Workplace app: advanced features like voicemail, presence updates, call transfer, and call recording
- Users may be confused about which app to use for which function
