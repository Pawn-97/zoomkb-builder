# Call Forwarding Obscures Original Caller ID

## Summary

When calls are forwarded to external numbers, the forwarding source number is displayed rather than the original caller's number. This is a significant UX consideration for call routing flows where preserving caller identity is important for recipients.

## Key points

- Forwarded calls show the forwarding number, not the original caller
- Applies to external forwarding destinations
- Cross-tenant forwarding between organizations also loses original caller info
- No configuration workaround mentioned in the article
