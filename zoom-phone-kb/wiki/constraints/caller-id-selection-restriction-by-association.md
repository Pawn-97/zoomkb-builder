# Caller ID Selection Restriction by Association

## Summary

Users can only select outbound caller IDs from numbers assigned to their own account, the company main number, or call queues where they are members. This restriction means the available set of caller ID options is dynamic based on the user's organizational relationships and can change as memberships are updated.

## Key points

- Only numbers from: user's account, company main number, or call queues the user belongs to
- Call queue numbers require explicit membership, not just assignment
- Numbers from call queues the user is not a member of are hidden from the dropdown
- Changes in call queue membership dynamically affect available caller ID options
