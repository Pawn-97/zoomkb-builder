# Client Version and Feature Compatibility Restrictions

## Summary

Peer-to-Peer media requires Zoom Workplace app version 6.7.0 or later, and is incompatible with certain features like recording or monitoring. Designers should consider how to communicate these incompatibilities — either by preventing toggling when incompatible features are active or by warning about the conflict.

## Key points

- Requires Zoom Workplace app version 6.7.0 or later
- Incompatible with recording or monitoring features being enabled
- Client and desk phone restart required after enabling/disabling
- Feature rolling out in phases over the next few months
