# Country-Specific Documentation Requirements

## Summary

Certain countries like Spain and Australia impose additional documentation requirements beyond the standard address verification. Designers should plan for country-specific conditional steps in the provisioning flow and surface these requirements at the right time — ideally before the admin commits significant effort.

## Key points

- Spain and Australia explicitly require additional documentation
- Country-specific requirements are listed at zoom.com/phone-number-requirements/
- Additional docs may include valid identification and proof of address
