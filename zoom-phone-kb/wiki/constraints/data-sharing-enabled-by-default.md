# Data Sharing Enabled by Default

## Summary

All available data is shared with Zoom Contact Center by default. This opt-out model means agents immediately see enriched information but admins must proactively disable data they want to restrict, which has implications for onboarding guidance and privacy-conscious organizations.

## Key points

- By default, all available data is shared with Zoom Contact Center
- Admins must proactively deselect checkboxes to restrict data sharing
