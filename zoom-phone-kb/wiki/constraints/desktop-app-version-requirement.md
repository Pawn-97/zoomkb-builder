# Desktop App Version Requirement

## Summary

Call summary templates require Zoom Workplace desktop app version 7.0.0 or later. Designers should consider graceful degradation or an upgrade prompt for users on older versions, as the setting may appear in the web portal but not function on outdated clients.

## Key points

- Windows: 7.0.0 or higher
- macOS: 7.0.0 or higher
- Linux: 7.0.0 or higher
- Minimum version: Zoom Workplace desktop app 7.0.0+
- Older clients may not respect the web portal's template configuration
- Consider upgrade nudge or version-awareness in the settings UI
