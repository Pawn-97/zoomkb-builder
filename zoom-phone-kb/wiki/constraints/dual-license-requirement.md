# Dual License Requirement

## Summary

Both Zoom Virtual Agent and Zoom Phone licenses are required to use the SMS tool integration. The UX must handle scenarios where one or both licenses are missing with clear upgrade guidance, and should not surface SMS configuration options if prerequisites are not met.

## Key points

- Zoom Virtual Agent license required
- Zoom Phone license required
- Both must be present for the SMS tool to be available
- Missing license should result in gated or hidden configuration options
