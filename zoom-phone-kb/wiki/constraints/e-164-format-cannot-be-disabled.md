# E.164 Format Cannot Be Disabled

## Summary

E.164 international formatting, including country codes, is always applied to caller ID and cannot be disabled or customized. This means local-format caller ID display is not achievable through Zoom Phone settings alone, and the actual display depends on the recipient's device behavior.

## Key points

- Country code prefixes are always included in caller ID
- No configuration option to display local format only
- Japanese +81 prefix specifically noted as immutable
- Display outcome depends on recipient's device, carrier, and settings
