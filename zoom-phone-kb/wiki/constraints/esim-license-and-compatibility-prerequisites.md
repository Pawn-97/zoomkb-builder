# eSIM License and Compatibility Prerequisites

## Summary

Users must have a qualifying Zoom Phone license, an assigned eSIM package, and a compatible Android/iOS device running the global minimum version of the Zoom mobile app. This multi-faceted prerequisite check should be surfaced clearly during setup to reduce support tickets.

## Key points

- Requires Zoom Phone Basic license, Zoom Workplace Pro Plus/Business Plus, or standalone calling plan
- Requires assigned Zoom Phone Mobile eSIM package from admin
- Zoom mobile app global minimum version or higher required
