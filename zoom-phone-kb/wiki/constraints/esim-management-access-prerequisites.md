# eSIM Management Access Prerequisites

## Summary

Accessing and operating eSIM Management requires account owner or admin privileges with permission to edit account settings. Additionally, each user being provisioned must have a Zoom Phone Mobile license, and their device must be a compatible eSIM-enabled smartphone.

## Key points

- Account owner or admin with edit account settings privilege
- Zoom Phone Mobile license required per user
- Compatible eSIM-enabled smartphone (iOS or Android) required
- Verified business address required for mobile eSIM numbers
- Navigation: Phone System Management > eSIM Management
- Landing page: tabular view with user, package, number, status, roaming, and actions columns
- Search bar at top for user lookup
- Ellipsis menu and action buttons per row
