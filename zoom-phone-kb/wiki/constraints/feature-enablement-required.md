# Feature Enablement Required

## Summary

The master-sub account number transfer feature is not on by default. Admins must contact Zoom Support to have it enabled on their account before the move UI becomes visible.

## Key points

- Must contact Zoom Support to enable
- UI options hidden until feature is turned on
- Both master and sub-accounts need Zoom Global Phone Number SKU
