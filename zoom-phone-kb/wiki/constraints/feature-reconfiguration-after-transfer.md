# Feature Reconfiguration After Transfer

## Summary

If the number had messaging services, SPAM monitoring, CNAM, branded calling, or similar features enabled before the move, those features must be reconfigured in the destination account. This includes creating sub-account-specific 10DLC brands and campaigns.

## Key points

- Design should warn admins about features that will be lost on transfer
- Messaging, SPAM, CNAM, branded calling all require re-setup post-transfer
- 10DLC campaigns may need to be recreated per sub-account
