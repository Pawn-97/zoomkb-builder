# File Upload Format and Size Constraints

## Summary

Uploaded documents must be in JPG, PNG, PDF, DOC, or DOCX format with a 10MB maximum size per file. Documents must be clear, legible, and show complete information with no cropped or partially visible content. These constraints require client-side validation with clear error messaging for each rule.

## Key points

- Accepted formats: JPG, JPEG, PNG, PDF, DOC, DOCX
- Maximum file size: 10MB per document
- Documents must be clear and legible (no automated check, relies on human review)
- No cropped or partially visible documents allowed
- All information on the document must be complete and visible
