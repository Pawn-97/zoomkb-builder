# License and Marketplace Prerequisites

## Summary

Users must have either a Zoom Workplace license with Phone or a standalone Zoom Phone calling plan. Additionally, Zapier must be added to the Zoom App Marketplace account before the integration can be used.

## Key points

- Zoom Workplace with Phone, or standalone Zoom Phone calling plan required
- Zapier must be pre-added to Zoom App Marketplace account
- App-wide triggers require User with Shared Access or Admin permission on Marketplace
