# License and Setup Prerequisites

## Summary

Call queue management requires either a Zoom Workplace license with Phone included or a standalone Zoom Phone calling plan. Additionally, the initial Zoom Phone setup must be completed before call queues can be created or managed.

## Key points

- Zoom Workplace with Phone, or standalone Zoom Phone calling plan required
- Initial Zoom Phone setup must be completed first
- Account owner or admin privileges required
