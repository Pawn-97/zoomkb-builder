# License-SIM Configuration Mismatch Detection

## Summary

The Zoom Phone Mobile license type (Primary vs. Secondary) must match the SIM configuration on the device. If a mismatch is detected, the user's data connectivity is automatically deactivated, enforcing the Acceptable Fair Use Policy.

## Key points

- Auto-detection of mismatch between license type and device SIM config
- Consequence: automatic data deactivation
- Design implication: clear in-product warnings during activation, not just policy text
