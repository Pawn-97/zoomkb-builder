# LMS Must Support LTI 1.3 with NRPS Scope

## Summary

Zoom Phone in LTI Pro only works with LMS platforms that support LTI 1.3 and have the Names and Role Provisioning Services (NRPS) scope enabled. This limits the feature to specific LMS configurations (Canvas, Moodle, Blackboard, Brightspace) and adds a prerequisite admin setup step before Zoom Phone can be enabled.

## Key points

- Only LTI 1.3 integrations are supported (not earlier LTI versions)
- NRPS scope must be enabled in the LMS before Zoom Phone integration
- Specific configuration steps exist for Canvas, Moodle, Blackboard, and Brightspace
