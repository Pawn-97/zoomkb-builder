# Multi-Step Enablement Prerequisites

## Summary

Before templates can be used, several prerequisites must be met: Custom AI Companion add-on, call summary with AI Companion enabled, call summary templates enabled, and desktop app 7.0.0+. This chain of dependencies should be visualized clearly so admins can see what's blocking template availability.

## Key points

- Requires: Zoom Phone license (or qualifying plan), Custom AI Companion add-on, admin privileges
- Call summary with AI Companion must be enabled
- Call summary templates feature must be enabled separately
