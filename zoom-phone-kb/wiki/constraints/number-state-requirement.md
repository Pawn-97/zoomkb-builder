# Number State Requirement

## Summary

Numbers must be in an unassigned state before they can be moved between accounts. Assigned numbers cannot be transferred.

## Key points

- Unassigned status is a hard prerequisite
- Design implication: need clear indication of which numbers are eligible for transfer
