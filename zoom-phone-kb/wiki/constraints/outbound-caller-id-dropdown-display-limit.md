# Outbound Caller ID Dropdown Display Limit

## Summary

The outbound caller ID dropdown is capped at 100 phone numbers maximum. For power users or organizations with many call queues, this means some assigned numbers may not be visible in the dropdown. Designers should consider how to handle overflow scenarios with search, filtering, or pagination within the selection UI.

## Key points

- Maximum of 100 phone numbers shown in the outbound caller ID dropdown
- Numbers beyond 100 are effectively invisible to the user
- No mention of search or filter within the dropdown to mitigate the limit
- Particularly impactful for users in many call queues
