# PAC Code 30-Day Expiry

## Summary

PAC codes obtained from current business mobile providers are valid for only 30 days. The porting workflow has a hard time constraint that, if missed, requires obtaining a new code from the provider, adding friction to the process.

## Key points

- PAC code expires 30 days after issuance
- Expired codes require obtaining a new one from the provider
- UX should display expiry countdown and send reminders
- Porting request must include both PAC code and expiry date
