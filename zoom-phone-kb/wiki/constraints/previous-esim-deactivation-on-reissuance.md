# Previous eSIM Deactivation on Reissuance

## Summary

Issuing a new eSIM automatically deactivates the previous eSIM. This is an important side effect admins must understand before reissuing, especially if the original device is not actually lost.

## Key points

- New eSIM issuance renders previous eSIM inactive
- Only one active eSIM per user at a time
- Design should clearly communicate this in the confirmation modal
