# Proof of Identity: Passport Only

## Summary

For the proof of identity document, only a valid passport is accepted. Other government-issued ID forms are explicitly not accepted. This is a stricter requirement than many other KYC processes and must be communicated prominently to avoid rejection and resubmission delays.

## Key points

- Only a valid passport is accepted for proof of identity
- Other ID forms (e.g., driver's license, national ID card) are not accepted
- This is stricter than many KYC processes users may be familiar with
- Must be communicated prominently in the upload interface
