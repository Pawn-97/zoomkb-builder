# QR Code Single-Use Limitation

## Summary

Each QR code can only be scanned and activated once. This means any issues during activation (e.g., interrupted process, wrong device) require generating a new QR code or using admin intervention, which impacts the error recovery UX.

## Key points

- QR codes are single-use: once scanned and installed, the code is invalidated
- Users can self-serve a new QR code for device upgrades
- Lost devices require admin to issue a New eSIM
