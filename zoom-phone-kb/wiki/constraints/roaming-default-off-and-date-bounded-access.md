# Roaming Default-Off and Date-Bounded Access

## Summary

International roaming is disabled by default for all new eSIM packages. Admins must explicitly enable it and set an end date. The day pass only activates when the user actually begins roaming, not at the time of admin configuration.

## Key points

- Default-off: admin must opt in per user
- End date required to limit unintended charges
- Day pass activated by user's actual roaming behavior, not admin action
- Design implication: clearly display roaming status and dates in user detail
