# Single Shared Line Group Membership

## Summary

A user can only belong to one shared line group at a time, and cannot be in both a shared line group and a call delegation setup simultaneously. This mutual exclusivity should be clearly communicated during group assignment to prevent confusion when a user tries to join a second group.

## Key points

- One shared line group membership per user
- Mutually exclusive with call delegation—cannot have both
- Users can check their current shared line group name via the Zoom Workplace app
