# Site-Scoped BYOC Number Restriction

## Summary

Zoom native numbers can only be forwarded to BYOC numbers assigned to users within the same Site being configured. This site-scoping constraint means the destination number selector must filter available BYOC numbers by site membership. The main company number is also excluded from forwarding configuration entirely.

## Key points

- BYOC destination numbers must be assigned to users within the same site
- Main company number cannot be configured as a forwarding source
- Source numbers must be associated with the site being configured
