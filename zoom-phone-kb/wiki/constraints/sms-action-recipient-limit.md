# SMS Action Recipient Limit

## Summary

The Zoom Phone for Zapier action can send SMS to a maximum of 10 recipients per Zap execution. This limit applies to the Send SMS action event.

## Key points

- Maximum 10 recipients per SMS action
- Design implication: validate recipient count in Zap configuration UI
