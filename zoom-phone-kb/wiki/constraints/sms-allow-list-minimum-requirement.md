# SMS Allow List Minimum Requirement

## Summary

At least one Zoom Phone user must be added to the SMS Allow List for the feature to function. The UX should validate this and prevent enabling SMS without at least one user, and should communicate why an Allow List is needed (so someone can view incoming customer replies).

## Key points

- At least one Zoom Phone user must be on the Allow List
- Allow List users are the only ones who can view incoming SMS replies
- Validation should prevent enabling SMS without at least one Allow List entry
