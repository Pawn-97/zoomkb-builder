# SMS/MMS App Restriction

## Summary

SMS and MMS are only supported through the mobile device's native messaging app. The Zoom Workplace app shows messages in read-only mode but cannot send them due to iOS and Android platform restrictions.

## Key points

- Native messaging app for sending SMS/MMS
- Zoom Workplace app: read-only message view
- Important UX implication: users need clear guidance on which app to use for which function
