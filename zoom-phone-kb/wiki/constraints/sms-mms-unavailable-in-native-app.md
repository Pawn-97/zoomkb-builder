# SMS/MMS Unavailable in Native App

## Summary

SMS and MMS are not supported by the mobile device's native messaging app for Zoom Phone eSIM numbers. The Zoom Workplace mobile app provides read-only messaging, but users cannot send messages from it due to iOS/Android ecosystem restrictions. Designers should set clear expectations to avoid user frustration.

## Key points

- No SMS/MMS support in native messaging app
- Zoom Workplace app offers read-only messaging view only
- Sending messages is blocked by iOS/Android ecosystem restrictions, not Zoom's design choice
