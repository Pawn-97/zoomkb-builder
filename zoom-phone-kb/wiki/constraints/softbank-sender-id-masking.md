# SoftBank Sender ID Masking

## Summary

For recipients on the SoftBank network, the actual phone number will not appear as the sender ID. Instead, SoftBank displays a unique sender ID assigned to each SMS aggregator. This carrier-imposed restriction is beyond Zoom's control, but businesses must be informed so they can set expectations with their customers about what sender ID will appear.

## Key points

- SoftBank recipients see a unique aggregator Sender ID, not the actual phone number
- This restriction applies to all A2P SMS from any aggregator on SoftBank
- Not configurable -- imposed by the carrier
