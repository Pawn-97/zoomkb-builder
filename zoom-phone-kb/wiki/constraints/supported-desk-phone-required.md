# Supported Desk Phone Required

## Summary

The star-code opt-in/out feature only works on supported desk phone models. Users with unsupported hardware cannot use this feature and must fall back to the Zoom Workplace app to manage their call participation. Designers should surface hardware compatibility information before the user attempts the star-code flow.

## Key points

- Only supported desk phone models are compatible with the *85 star code
- Unsupported phones require the Zoom Workplace app as fallback
