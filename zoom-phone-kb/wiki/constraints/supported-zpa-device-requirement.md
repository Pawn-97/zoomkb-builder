# Supported ZPA Device Requirement

## Summary

Only supported Zoom Phone Appliance devices can be monitored through this alerting system. Unsupported or third-party devices are not eligible for alert configuration and should be excluded from the device selection interface.

## Key points

- Monitoring only works with supported Zoom Phone Appliances
- Unsupported devices should be excluded from target selection
