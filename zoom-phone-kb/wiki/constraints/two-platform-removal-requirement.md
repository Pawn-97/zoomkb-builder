# Two-Platform Removal Requirement

## Summary

Fully disconnecting the integration requires actions on both Zapier and Zoom platforms. Simply removing the connection on one side leaves residual configuration on the other. This cross-platform dependency may confuse users expecting a single removal action.

## Key points

- Removal requires actions on both Zapier (Manage Connections) and Zoom (App Marketplace)
- No single-click disconnect across platforms
- Design implication: provide clear documentation and in-product guidance for the two-step process
