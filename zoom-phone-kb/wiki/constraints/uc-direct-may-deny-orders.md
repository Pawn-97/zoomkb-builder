# UC Direct May Deny Orders

## Summary

UC Direct reserves the right to reject hardware orders. Zoom can review software-related cases if applicable, but hardware order denial is ultimately UC Direct's decision. The UI should set appropriate expectations and provide recourse paths if an order is denied.

## Key points

- UC Direct has independent authority to reject hardware orders
- Zoom can review cases only if they involve the software portion of the order
