# UI Visibility for Master/Sub Classification

## Summary

The move-to-account option should only be visible when the account relationship is a master-sub classification. This implies conditional rendering of UI elements based on account hierarchy.

## Key points

- Conditional visibility based on account relationship type
- Non-hierarchical accounts should not see the move option
