# Voice Agent Type Restriction

## Summary

This SMS tool configuration applies specifically to voice agents in Zoom Virtual Agent. Chat agents and classic chatbots use different setup and deployment methods. Designers should ensure the SMS configuration interface is only surfaced for voice agent types to avoid confusion.

## Key points

- SMS tool is specific to instruction-based voice agents
- Chat agents and classic chatbots have separate setup processes
- The SMS configuration should only appear for applicable agent types
- Different agent types may have different documentation references
