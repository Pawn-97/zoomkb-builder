# Zoom Desktop Client Required for Call Routing

## Summary

The Zoom Phone widget in LTI Pro uses the Zoom desktop client to route calls. Users cannot make or receive calls through LTI Pro without the Zoom client installed and signed in. This creates a two-application dependency that should be clearly communicated in the UI.

## Key points

- Zoom desktop client must be installed and signed in
- Same Zoom user must be signed into both LTI Pro and the Zoom client
- Without the client, the Phone button in LTI Pro is non-functional
