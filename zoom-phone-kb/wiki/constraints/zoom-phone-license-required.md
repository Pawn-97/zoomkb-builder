# Zoom Phone License Required

## Summary

A Zoom Phone license is a prerequisite for using the shared line group opt-in/out feature. Without a license, the star code will not be recognized. This is a gate that should be validated before exposing the feature to users.

## Key points

- Zoom Phone license is required for the user
- Feature is non-functional without the license
