# ZPLS Account and Subscription Requirements

## Summary

Only Business, Education, or Enterprise accounts with both Zoom Node and Zoom Phone subscriptions can use ZPLS data synchronization. The admin must also have a deployed ZPLS service module. The UI should gate access to sync features behind these requirements.

## Key points

- Business, Education, or Enterprise account required
- Zoom Node and Zoom Phone subscriptions required
- ZPLS service module must be deployed before sync is available
