# ZPLS and Account Prerequisites

## Summary

Call Bridge deployment requires Business/Education/Enterprise account, Zoom Node and Zoom Phone subscriptions, admin privileges, and a deployed ZPLS service module. These are the same prerequisites as other ZPLS features, suggesting a unified account eligibility check should be implemented across the admin interface.

## Key points

- Business, Education, or Enterprise account
- Zoom Node and Zoom Phone subscriptions
- ZPLS service module must already be deployed
- Account owner or admin privileges required
