# ZPLS Prerequisites for Node Groups

## Summary

Node groups require Business/Education/Enterprise account, Zoom Node and Zoom Phone subscriptions, a deployed ZPLS service, ZPLS enabled, and a pre-configured SBC with ZPLS. This creates a dependency chain where node groups are one of the last things configured in the ZPLS setup sequence.

## Key points

- Business, Education, or Enterprise account required
- Zoom Node and Zoom Phone subscriptions required
- ZPLS service must be deployed and enabled
- SBC must be pre-configured with ZPLS (dependency on KB0085804)
