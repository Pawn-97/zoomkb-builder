# Add a Call Queue Number to a Phone User's Profile

## Summary

Admins add a specific call queue phone number to a user's outbound caller ID options via the user's profile. This can include additional custom numbers beyond the standard set. The flow requires navigating to the user rather than the call queue, which means admins may need to switch between different management interfaces to fully configure caller ID.

## Key points

- Path: Phone System Management > Users & Rooms > Users > [user] > Profile > Outbound Caller ID
- Select call queue number from dropdown, then Save
- Optional 'Customize Numbers > Add' for additional phone numbers
- Both account numbers and call queue numbers can be added as custom options
