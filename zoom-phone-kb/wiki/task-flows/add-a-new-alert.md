# Add a New Alert

## Summary

A guided configuration flow with multiple interdependent form sections. The rule selection drives which target options and parameters appear, creating conditional form behavior. The flow separates configuration into Rule, Notification, and Active Status sections, providing a structured but potentially complex form experience.

## Key points

- Click Add an alert to enter the creation page
- Under Rule: set Name, ensure Module is ZPA Management, select a Rule type, and configure Target based on rule selection
- Under Notification: set Frequency, Time frame, Email Recipients (up to 10), and optionally Chat Channels (up to 3)
- Under Active Status: toggle to enable or disable (defaults to active)
- Click Save to create
