# Add Users as Members of a Call Queue

## Summary

Admins add users to a call queue so those users can select the queue's number as their outbound caller ID. The flow goes through Phone System Management > Call Queues > [queue] > Profile > Members, with search and add functionality. The UX challenge is that the relationship between call queue membership and caller ID availability may not be intuitive to admins.

## Key points

- Path: Phone System Management > Call Queues > [queue] > Profile > Members > View or Edit
- Search for existing members or add new users via search box
- Users tab available for adding users not currently in the queue
- Changes take effect immediately; no additional save step for membership
