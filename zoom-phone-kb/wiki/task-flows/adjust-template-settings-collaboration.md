# Adjust Template Settings (Collaboration)

## Summary

Admins configure the collaboration mode for a template via a Settings icon panel. Choose between Everyone (open editing) and Restricted (creator + named collaborators). When Restricted is selected, an Add Users flow appears to populate the Collaborators list. This panel-based settings pattern separates collaboration controls from content editing.

## Key points

- Click Settings icon on the template page to open a settings panel
- Select 'Everyone' for open editing or 'Restricted' for limited access
- Under Restricted, add users to the Collaborators list
- Collaborators appear in a managed list with presumably add/remove capabilities
