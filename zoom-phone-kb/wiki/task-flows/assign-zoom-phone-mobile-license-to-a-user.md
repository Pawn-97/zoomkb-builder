# Assign Zoom Phone Mobile License to a User

## Summary

A two-stage assignment flow: first, the admin assigns at least a Zoom Phone Basic license to the user in User Management; then, they assign the Zoom Phone Mobile license in eSIM Management. This split reflects the different lifecycle domains (user identity vs. mobile connectivity).

## Key points

- Stage 1: Assign Zoom Phone Basic license in User Management
- Stage 2: Assign Zoom Phone Mobile license in eSIM Management
- First-time assignment auto-generates an eSIM and sends QR code email to user
- Mobile device endpoint auto-created in user's call handling settings
