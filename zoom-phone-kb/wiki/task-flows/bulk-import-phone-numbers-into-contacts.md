# Bulk Import Phone Numbers into Contacts

## Summary

A CSV-based import flow for populating the entire course roster with phone numbers at once. The downloadable template ensures format consistency. This is a classic bulk-operation pattern: download template, edit offline, upload, confirm.

## Key points

- Click Import Phone Numbers in the Contacts tab
- Download the CSV template
- Edit the CSV with phone numbers
- Upload the edited file via the import dialog and click Upload
