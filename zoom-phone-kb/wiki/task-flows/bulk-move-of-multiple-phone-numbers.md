# Bulk Move of Multiple Phone Numbers

## Summary

Admins can select multiple numbers at once using checkboxes (including a select-all option) and initiate a batch transfer. This flow includes address verification if any native numbers are in the selection.

## Key points

- Uses checkbox selection with a select-all shortcut in the column header
- Dedicated toolbar button: 'Move numbers between master/sub account'
- Address verification pop-up appears if native numbers are selected
- Supports moving native, BYOC, or mixed number types in one batch
