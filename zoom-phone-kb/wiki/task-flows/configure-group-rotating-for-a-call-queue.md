# Configure Group Rotating for a Call Queue

## Summary

Admins enable group rotating distribution on an existing call queue by navigating to the queue's Profile tab, selecting Group Rotating from the Call Distribution dropdown, and setting the group size and ringing duration. Overflow behavior must also be configured before saving.

## Key points

- Navigate to Call Queues, click queue name, then Profile tab
- Dropdown selection under Business Hours: change from default to Group Rotating
- Two numeric inputs: Group member size and Ringing duration per group
- Requires explicit Confirm action before Save
- Overflow configuration is a recommended follow-up step before saving
