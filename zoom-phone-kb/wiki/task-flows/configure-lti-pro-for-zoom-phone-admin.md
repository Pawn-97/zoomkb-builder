# Configure LTI Pro for Zoom Phone (Admin)

## Summary

Admins enable Zoom Phone for an existing LTI 1.3 integration through the Zoom Marketplace. This is a single toggle within the LTI Pro app configuration. The flow assumes the LTI 1.3 credential is already set up; if not, the admin must configure that first before the Zoom Phone toggle becomes available.

## Key points

- Navigate to Zoom Marketplace > Manage > Apps on Account > LTI Pro App
- Open Configure and select the LTI 1.3 credential
- Enable the Zoom Phone toggle
- LTI 1.3 credential must exist before the toggle appears
