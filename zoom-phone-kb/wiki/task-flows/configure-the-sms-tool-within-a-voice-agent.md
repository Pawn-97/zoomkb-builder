# Configure the SMS Tool Within a Voice Agent

## Summary

Admins create or edit a voice agent, define a skill with trigger description, then insert the SMS tool into the instruction using search or forward-slash. They configure the sender number (entry point), recipient phone number variable, and SMS content with dynamic variables. The flow ends with Save and Publish. The UX challenge is making tool configuration accessible to admins who may not be familiar with variable-driven message composition.

## Key points

- Start from an existing or new instruction-based voice agent
- Define a skill name and trigger description in the Skills section
- Insert SMS tool via search or forward-slash (/) shortcut
- Configure: sender number (entry point), recipient number (variable), message content (with dynamic variables)
- Save and add, then publish the voice agent to apply changes
