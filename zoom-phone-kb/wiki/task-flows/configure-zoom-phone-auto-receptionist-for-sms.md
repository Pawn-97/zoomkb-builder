# Configure Zoom Phone Auto Receptionist for SMS

## Summary

Admins navigate to Phone System Management > AI Receptionist > Virtual Agent, select the associated auto receptionist, assign a 10-digit phone number with an approved messaging campaign, enable the SMS toggle under Policy, and add Zoom Phone users to the Allow List. Each step has dependencies (number needs campaign, toggle needs Allow List) that must be validated in sequence.

## Key points

- Path: Phone System Management > AI Receptionist > Virtual Agent tab
- Select auto receptionist associated with the voice agent
- Assign a 10-digit phone number (must have Approved Messaging Campaign)
- Enable SMS toggle under Policy > General
- Add at least one Zoom Phone user to the Allow List for SMS replies
