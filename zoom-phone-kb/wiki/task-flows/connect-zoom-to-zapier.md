# Connect Zoom to Zapier

## Summary

The initial setup flow where a user adds Zoom Phone for Zapier as a connection in Zapier's platform. The user searches for the Zoom Phone app, initiates the connection, authenticates via a browser-based OAuth flow, and optionally enables shared access for organization-wide triggers.

## Key points

- Access Zapier Apps page, click Add connection
- Search and select Zoom Phone for Zapier
- Browser-based OAuth: new tab opens for Zoom sign-in and permission grant
- Shared Access checkbox available during authentication
- Once connected, the app appears in Zapier for use in Zaps
