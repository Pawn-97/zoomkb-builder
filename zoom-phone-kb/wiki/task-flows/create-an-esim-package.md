# Create an eSIM Package

## Summary

Admins create a new eSIM package by selecting a user, a package tier, and a phone number from dropdowns in a pop-up modal. On save, the system auto-provisions the subscriber, generates eSIM details, and sends a welcome email with a QR code to the user.

## Key points

- Accessed via Create button in top-right corner of eSIM Management
- Three dropdown selections: Username, Package, Number
- Phone number must already exist in the account
- Auto-provisioning on save: subscriber created, eSIM generated, QR email sent
