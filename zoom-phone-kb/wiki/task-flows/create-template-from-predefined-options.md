# Create Template from Predefined Options

## Summary

Admins select from five predefined template types (General, One-on-one, Project update, Discussion, Customer success), each designed for a specific call scenario. After selecting a type, admins configure icon, name, description, and user assignment. This flow is lower-friction than creating from scratch and helps admins get started quickly with battle-tested formats.

## Key points

- Click '+ Create from template', select from five predefined options in a pop-up
- Five templates: General (recap + next steps), One-on-one (recap of every topic), Project update (progress + next steps), Discussion (key points overview), Customer success (feedback + next steps)
- Configure icon, name, optional description, and 'Who can use this?' assignment
- Save creates the template and makes it available for further customization
