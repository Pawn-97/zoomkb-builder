# Create Template from Scratch

## Summary

Admins use an AI prompt interface to generate a completely custom template by describing the desired format, sections, key information, and industry-specific terminology. After generation, the template can be reviewed, adjusted, and saved. This natural-language-to-template workflow is a novel UX pattern that puts significant weight on prompt quality and review affordances.

## Key points

- Click 'Create from scratch', enter a detailed prompt describing format, sections, highlights, and terminology
- Click 'Create template' to generate via AI
- Review the generated template and adjust icon, name, description, and user assignment
- Save to make it available for editing and customization
