# Customize Existing Template Content

## Summary

Admins modify the actual template formatting and structure: adjusting length, adding/removing sections, changing terminology, and setting the level of detail. This is the content-authoring part of template management and requires direct manipulation of the AI-generated structure.

## Key points

- Select a template, then modify content directly
- Adjust formatting: select Length, add/remove sections, change terminology and detail level via text boxes
- Use '+ Add a section' to expand the template structure
