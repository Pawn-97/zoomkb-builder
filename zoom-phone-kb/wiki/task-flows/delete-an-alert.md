# Delete an Alert

## Summary

Two deletion paths are available: an inline action via the ellipsis menu on the alert list, or a delete button at the bottom of the alert detail page. Both paths include a confirmation dialog to prevent accidental deletion.

## Key points

- Option A: Click the ellipsis next to the alert name in the list and select Delete
- Option B: Open the alert, scroll to the bottom, and click Delete
- Confirm deletion in the confirmation dialog
