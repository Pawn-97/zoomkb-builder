# Deploy Call Bridge Service

## Summary

A modal-based workflow where admins add the Call Bridge service on a Node server. The flow includes selecting the target Node server, providing the internal IP, and optionally setting an internal domain prefix. The step-by-step dialog approach minimizes confusion for this one-time setup task, but the IP address field requires knowledge of the initial Node server configuration.

## Key points

- Navigate: Node Management > Modules > Phone Local Survivability > Services tab > Add Services > Call Bridge
- Select target Node server, enter internal IP from initial Node configuration
- Optional internal domain prefix
- Modal dialog guides admin through service addition
