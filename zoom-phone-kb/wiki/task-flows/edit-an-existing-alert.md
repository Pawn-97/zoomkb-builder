# Edit an Existing Alert

## Summary

Admins can modify any aspect of a configured alert. The rename action is separate from editing rule, target, and notification parameters. Email recipients can be added but the article does not describe a removal mechanism, which may be a UX gap for alert management.

## Key points

- Click the alert name to open its detail/edit view
- Rename via a separate Rename button next to the alert name
- Edit Rule, Target, and Time frame via individual Edit buttons
- Add email recipients via Add button
- Toggle Active Status to enable or disable the alert
