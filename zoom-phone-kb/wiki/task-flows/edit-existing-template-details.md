# Edit Existing Template Details

## Summary

Admins modify template metadata (icon, name, description, user assignment) via a pencil icon-triggered pop-up. Templates can also be deleted from this view. The pencil icon edit pattern separates metadata editing from content customization, helping admins understand which changes affect template content vs. template management.

## Key points

- Click a template, then click the pencil icon
- Edit icon, name, optional description, and 'Who can use this?' assignment
- Option to delete the template with a confirmation dialog
- Save to apply changes
