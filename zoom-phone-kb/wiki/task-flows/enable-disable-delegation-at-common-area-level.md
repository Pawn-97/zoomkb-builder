# Enable/Disable Delegation at Common Area Level

## Summary

Controls delegation for common area phones (shared devices like lobby or conference room phones). The flow mirrors the phone user level but with a simplified set of options (no site-restriction checkbox). Navigation is through the Common Areas tab next to Users under Users & Rooms, providing a parallel but distinct path.

## Key points

- Navigate to Phone System Management > Users & Rooms > Common Areas tab > select common area > Policy tab
- Toggle Delegation under the General section
- No site-restriction option (unlike phone user level)
- Changes apply only to the selected common area
