# Enable/Disable Delegation at Group Level

## Summary

Allows admins to control delegation for a specific user group. If the account-level setting is locked, the group toggle appears grayed out. The navigation path (User Management > Groups) differs from the account path, which could cause confusion -- designers should consider consistent navigation patterns for settings that span hierarchy levels.

## Key points

- Navigate to User Management > Groups > select group > Zoom Phone tab
- Toggle Delegation under the General section
- Grayed out if locked at account level
- Optional: lock to prevent user-level changes within the group
