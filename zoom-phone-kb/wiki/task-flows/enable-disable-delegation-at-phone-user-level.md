# Enable/Disable Delegation at Phone User Level

## Summary

The most granular delegation control, applied to individual phone users. Includes an additional option not present at higher levels: 'Restrict from adding delegation to users outside their own site', which limits cross-site delegation. This site-boundary constraint is a privacy/governance feature that only appears at the user level and should be clearly explained.

## Key points

- Navigate to Phone System Management > Users & Rooms > Users tab > select user > Policy tab
- Toggle Delegation under the General section
- Additional option: Restrict from adding delegation to users outside their own site (checkbox)
- Verification dialog confirms each change
