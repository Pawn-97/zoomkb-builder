# Enable/Disable Delegation at Site Level

## Summary

Controls delegation for users within a specific site. The navigation path (Phone System Management > Company Info > Policy tab) differs from both Account and Group paths, introducing a third admin navigation pattern for the same feature. Consistent Policy tab placement at the Site level provides some predictability.

## Key points

- Navigate to Phone System Management > Company Info > select site > Policy tab
- Toggle Delegation under the General section
- Grayed out if locked at account level
- Optional: lock to prevent user-level changes within the site
