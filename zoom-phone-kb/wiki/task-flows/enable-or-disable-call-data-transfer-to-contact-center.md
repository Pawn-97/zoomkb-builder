# Enable or Disable Call Data Transfer to Contact Center

## Summary

A simple single-checkbox toggle buried four levels deep in the admin navigation. The setting is labeled 'Transferring call data to Contact Center' but the actual control is a checkbox for 'Conversation summary,' which creates a potential terminology mismatch. The flow is a basic enable/disable with no additional configuration or parameters.

## Key points

- Sign in to Zoom web portal as an admin with edit privileges
- Navigate to Phone System Management > Company Info > Account Settings
- Locate 'Transferring call data to Contact Center' under the Preference section
- Check 'Conversation summary' to enable, uncheck to disable
