# Enable or Disable Peer-to-Peer Media for Individual Phone User

## Summary

Individual user configuration through Phone System Management > Users & Rooms. When grayed out, the note explicitly tells the admin the setting is locked at group or account level and advises contacting the Zoom admin, providing clear guidance on next steps.

## Key points

- Navigate: Phone System Management > Users & Rooms > Users tab > select user > Policy tab
- Toggle Peer to Peer Media under General section
- Option grayed out if locked at group or account level
- Grayed-out note instructs admin to contact higher-level admin
