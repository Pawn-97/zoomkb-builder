# Generate New QR Code (Device Upgrade)

## Summary

Users upgrading to a new device can regenerate their QR code from the web portal without admin intervention, provided they still have the old device to remove the eSIM. For lost devices, only admins can issue a new eSIM. This self-service vs. admin-assisted distinction is a key UX decision point that affects user autonomy and support burden.

## Key points

- Available through Phone > Settings > eSIM > View QR Code in the web portal
- Only for device upgrades where the old device is still accessible to remove the eSIM
- For lost devices, only admins can use the 'New eSIM' option
