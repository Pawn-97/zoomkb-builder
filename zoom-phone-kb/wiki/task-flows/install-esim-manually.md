# Install eSIM Manually

## Summary

Fallback activation path for users who cannot scan the QR code. Users copy SM-DP+ address and activation code from the activation message into their device's cellular settings. The manual path requires text entry of long codes, so clear copy affordances and error-tolerant input fields are important UX considerations.

## Key points

- On iOS: Settings > Cellular > Set Up Cellular Plan > enter SM-DP+ address and activation code
- On Android: Settings > Network & Internet > Mobile network > Add SIM > enter network provider code
- Activation code expires once used
- Android steps may vary by device model
