# Install eSIM via QR Code

## Summary

Users scan the QR code from the activation message using their camera app or SIM management settings, then follow on-device prompts to complete activation. This flow partially leaves the Zoom ecosystem (device-native steps), so handoff UX and clear in-context instructions are critical.

## Key points

- Scan QR code using camera app or SIM management settings
- Tap 'Add Cellular Plan' and follow device manufacturer instructions
- QR code is single-use; once scanned and installed, it becomes invalid
