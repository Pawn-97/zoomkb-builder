# Issue a New eSIM (Reissuance)

## Summary

When a user loses their device or an eSIM needs replacement, admins click 'Issue new eSIM' from either the list view or detail page. A confirmation modal warns that the previous eSIM will be deactivated. On confirm, a new eSIM is generated and a welcome email with new QR code is sent.

## Key points

- Two access points: Action column in list view, or action bar in detail page
- Confirmation modal with warning about previous eSIM deactivation
- Triggers new QR code email to user
- Use case: lost/stolen devices, or eSIM regeneration
