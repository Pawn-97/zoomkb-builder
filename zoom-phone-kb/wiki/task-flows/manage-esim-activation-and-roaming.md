# Manage eSIM Activation and Roaming

## Summary

Admins manage the eSIM lifecycle from eSIM Management, including issuing first-time eSIMs (automated on license assignment), reissuing eSIMs for lost devices, and deactivating/activating service. International roaming is controlled per user and is disabled by default to prevent unexpected charges.

## Key points

- Auto-issuance on first license assignment
- Reissue for lost/stolen devices
- Deactivate/activate service per user
- Roaming disabled by default; admin must explicitly enable
- Day-pass roaming model: only active when user actually roams
