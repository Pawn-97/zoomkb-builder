# Manage International Roaming

## Summary

Admins control roaming for each user from the user detail page. They can add a new roaming plan with start and end dates, edit existing plan dates, or delete active roaming service. Roaming is disabled by default for all new packages.

## Key points

- Add roaming: click Add International Roaming, set start/end dates, confirm
- Edit roaming: click Edit on existing entry, modify dates, confirm
- Delete roaming: ellipsis > Delete on existing entry, confirm deletion
- Date picker UI for start/end dates
- Day pass only activates on actual roaming
