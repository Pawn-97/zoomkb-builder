# Move a Single BYOC Phone Number

## Summary

Similar to the native number flow but simpler, as no business address verification is needed. Admins fill in the destination account, product, subscription, division, and SIP group, then confirm the move in a single wizard step.

## Key points

- Same entry point via ellipsis menu on Phone Numbers page
- Five dropdown fields but no business address step
- Single confirmation (Move button) completes the transfer
- Simpler UX reflects fewer constraints compared to native numbers
