# Open Zoom Phone in LTI Pro (Daily Use)

## Summary

After initial authorization, the Phone button in LTI Pro opens the Zoom Phone floating window. Users must be signed into the Zoom desktop client beforehand. This two-app dependency (LMS browser + Zoom client) creates a persistent setup step that designers should consider surfacing as a pre-flight check.

## Key points

- Sign in to Zoom desktop client with the same Zoom user account
- Click the Phone button in LTI Pro to open the floating phone window
