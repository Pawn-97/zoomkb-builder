# Remove Zapier for Zoom Phone App

## Summary

A cross-platform disconnection flow that requires actions in both Zapier and Zoom. In Zapier, the user deletes connected Zoom Phone accounts from Manage Connections. In the Zoom App Marketplace, the user navigates to Manage > Added Apps and removes the Zapier for Phone app.

## Key points

- Two-platform removal: Zapier (Manage Connections) and Zoom (App Marketplace > Added Apps)
- Zapier side: delete connected Zoom Phone accounts
- Zoom side: navigate to Manage > Added Apps, find Zapier for Phone, click Remove, confirm
- Sequential dependency: Zapier disconnection should happen first
