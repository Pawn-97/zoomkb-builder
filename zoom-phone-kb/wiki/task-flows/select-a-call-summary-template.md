# Select a Call Summary Template

## Summary

After enabling the feature, admins or users select from predefined or custom templates. The UX must differentiate between system-provided predefined templates and custom admin-created templates, with clear labeling so users understand the origin and purpose of each option.

## Key points

- Two template categories: predefined (system-provided) and custom (admin-created)
- Available at every configuration level (account, group, site, user, personal)
- Template selection is optional but likely to be the primary action after enabling
