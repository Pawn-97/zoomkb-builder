# Toggle Shared Line Group Call Participation via IP Phone

## Summary

A two-step audible-only flow: the user picks up the handset, dials a specific code, and hears a spoken confirmation. There is no screen interaction, so the entire UX is auditory. Designers should ensure the voice prompt is clear, concise, and unambiguously communicates the new state (opted in vs. opted out).

## Key points

- Pick up the IP phone handset
- Dial *85 followed by the shared line group extension number
- Listen for the confirmation prompt: 'You are now opted out' or 'You are now opted in'
