# View and Manage User eSIM Details

## Summary

Admins drill into a specific user's eSIM details by clicking the ellipsis menu and selecting View. The detail page shows General Info (user name, package, number, status, roaming details) and provides action buttons at the top (Issue new eSIM, Deactivate, Enable).

## Key points

- Access via ellipsis > View on the eSIM Management list
- Detail page organized into action bar and General Info section
- Editable fields: user name, phone number (Edit buttons inline)
- Roaming details sub-panel with Add, Edit, Delete actions
