# Account Owner / Admin (Call Data Sharing)

## Summary

Configures the Transferring Call Data policy for both Zoom Phone and Zoom Contact Center. Admins set which data fields are shared, manage lock settings to prevent downstream changes, and can optionally configure CRM variables, custom variables, and address book custom fields for Contact Center transfers.

## Key points

- Configures policies in two separate admin areas: Phone System Management > Company Info and Contact Center Management > Preferences
- Can lock settings at the account level to prevent users from overriding them
- Must have privilege to edit account settings
