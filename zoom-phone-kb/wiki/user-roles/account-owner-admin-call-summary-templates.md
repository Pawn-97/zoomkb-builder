# Account Owner / Admin (Call Summary Templates)

## Summary

Creates, edits, customizes, tests, publishes, and manages call summary templates. Has full control over template lifecycle, collaboration settings, and user assignment. Must navigate to AI Studio (not Phone settings) to manage templates, which may not be immediately intuitive for phone-focused admins.

## Key points

- Full lifecycle control: create, edit, customize, test, publish, manage templates
- Sets 'Who can use this?' assignment and collaboration mode
- Accesses templates through AI Companion > AI Studio (not Phone settings)
