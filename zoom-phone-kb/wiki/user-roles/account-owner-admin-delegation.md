# Account Owner / Admin (Delegation)

## Summary

Enables or disables the call delegation feature at any hierarchy level. Must have specific privilege scopes depending on the level being configured: account settings for Account level, group management for Group level, and site management for Site level. The UI must clearly indicate which privilege is needed at each level and provide appropriate access controls.

## Key points

- Account level: requires privilege to edit account settings
- Group level: requires privilege to edit groups
- Site level: requires privilege to edit sites
- Can lock settings to prevent lower-level overrides
- Manages delegation through different navigation paths depending on hierarchy level
