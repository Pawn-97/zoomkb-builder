# Account Owner / Admin (eSIM)

## Summary

Responsible for issuing Zoom Phone Mobile eSIM packages to users and generating replacement QR codes or new eSIMs for lost devices. The admin interface separates two important actions: 'New QR code' for device upgrades where the old device is still accessible, and 'New eSIM' for lost devices -- a distinction that must be clear in the UI to prevent accidental service disruption.

## Key points

- Issues eSIM packages and generates activation QR codes for users
- Has two distinct recovery paths: New QR code for upgrades vs. New eSIM for lost devices
- Manages eSIMs through Phone System Management in the web portal
