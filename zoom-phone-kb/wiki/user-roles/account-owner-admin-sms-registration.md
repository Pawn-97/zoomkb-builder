# Account Owner / Admin (SMS Registration)

## Summary

Manages the entire Japan SMS registration process, from creating business registrations to submitting SMS registrations for carrier approval. Must have SMS setup privileges in addition to admin access. This role navigates a multi-step workflow with external dependencies, so the interface should clearly show process status and required actions at each stage.

## Key points

- Must be account owner or admin with SMS setup privileges
- Manages business registration lifecycle (create, update, delete when allowed)
- Creates SMS registrations, configures sample messages, assigns numbers, manages lifecycle states
- Accesses SMS registration through Number Management > SMS Setup
