# Account Owner or Admin

## Summary

The only role with privileges to manage phone numbers and complete business address verification. Admins encounter a gated workflow where multiple prerequisites (address verification, document upload, additional country-specific requirements) must be satisfied before being able to select and purchase numbers.

## Key points

- Receives automatic device model corrections across the web portal
- Sees persistent red warnings for non-certified devices
- Responsible for device fleet management and deployment accuracy
- Can configure at account, group, site, and individual user levels
- Can lock settings to prevent changes at lower hierarchy levels
- Requires edit privileges for the relevant scope (account settings, groups, etc.)
- Must have a Zoom Workplace license with Phone or standalone Zoom Phone calling plan
- Must have privilege to edit account settings and manage phone numbers
- Responsible for uploading business verification and tax documents
- Receives email notifications about document review status
- Must navigate both Zoom web portal and Provider Exchange portal
- Must have account owner or admin privileges
- Configures the setting at the account level only
- Requires Zoom Workplace license with Phone or standalone Zoom Phone calling plan
- Setting applies to all users in the account with no per-user override
