# Account Owner

## Summary

The sole role authorized to submit business address verification and manage document uploads for Zoom Phone Mobile UK. Admins navigate the verification workflow, upload sensitive business documents, manage temporary numbers, and assign numbers to users. The UX must accommodate the responsibility of handling legal documents with clear guidance on requirements, formats, and acceptance criteria.

## Key points

- Can transfer numbers between any related master and sub-accounts
- Must have manage sub-accounts privilege enabled
- Responsible for moving numbers from sub-accounts back to master before deleting sub-accounts
- Full access to Phone System Management > Call Queues
- Can configure all distribution methods and overflow settings
- Requires Zoom Workplace license with Phone or standalone Zoom Phone plan
- Can purchase and assign Zoom Phone Mobile licenses
- Manages eSIM issuance and international roaming
- Receives roaming control toggles per user
- Automatically issued eSIM on first license assignment
- Full access to eSIM Management under Phone System Management
- Can create, view, edit, deactivate, and reissue eSIM packages
- Manages international roaming plans (add, edit, delete)
- Requires privilege to edit account settings
- Must have account owner or admin privileges with permission to edit account settings
- Requires a Zoom Phone license
- Access to alert configuration is via Account Management > Alerts & Notifications in the web portal
- Must have account owner or admin privileges to edit account settings
- Requires Zoom Workplace license with Phone included, or standalone Zoom Phone calling plan
- Setting is not visible or editable by non-admin users
- Can configure at account, group, site, and individual phone user levels
- Can lock settings to prevent changes by lower-level roles
- Can select default call summary templates for each level
- Must have Custom AI Companion add-on and call summary enabled
- Configures auto-receptionist caller ID settings for users
- Adds and removes users from call queues to enable caller ID options
- Manages outbound caller ID at the profile level for individual phone users
- Troubleshoots call failures related to caller ID misconfiguration
- Submits business address verification and uploads documentation
- Manages temporary numbers and assigns them to users
- Must have privilege to edit account settings
- Acts as the intermediary between Zoom verification and end users
- Needs clear feedback on document acceptance, rejection, and resubmission
- Configures SMS tool within the voice agent's instruction builder
- Sets up Zoom Phone Auto Receptionist SMS policy and Allow List
- Assigns 10-digit phone numbers with approved messaging campaigns
- Must have privileges for both Zoom Virtual Agent and Zoom Phone settings
