# Admin

## Summary

Admins can perform the same Zoom Phone Mobile management actions as account owners: assign licenses, issue/reissue eSIMs, manage roaming, and deactivate/activate service. Requires appropriate admin privileges for eSIM Management access.

## Key points

- Requires explicit manage sub-accounts privilege
- Must have appropriate product licenses (Zoom Phone or Zoom Contact Center)
- Can perform individual, bulk, and CSV transfer operations
- Access to Call Queues management interface
- Can configure group rotating and overflow settings
- Requires initial Zoom Phone setup completion
- Access to eSIM Management for license assignment and eSIM operations
- Can control roaming access and deactivate/activate user service
- Same capabilities as account owner within eSIM Management scope
- Must have privilege to edit account settings
- Full eSIM Management capabilities matching account owner
- Requires Zoom Phone Mobile license per user to provision
