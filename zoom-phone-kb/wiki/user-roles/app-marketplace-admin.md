# App Marketplace Admin

## Summary

A role on the Zoom App Marketplace with Admin permissions, allowing the user to add and configure the Zapier for Zoom Phone app and grant shared access for all-account triggers. This role manages the connection between Zoom and Zapier at the account level.

## Key points

- Admin permission on Zoom App Marketplace required for shared access triggers
- Manages app connection and removal from the Marketplace
- Can authenticate Zapier and grant permissions on behalf of the organization
