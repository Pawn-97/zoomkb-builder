# App Marketplace User with Shared Access

## Summary

A user role on the Zoom App Marketplace with Shared Access permission, which grants the ability to trigger the integration for all users in the account. This is an alternative to Admin permission for enabling organization-wide Zap triggers.

## Key points

- User role with Shared Access permission on App Marketplace
- Can enable all-user trigger scope without full admin rights
- Must be added to Zoom App Marketplace with this specific permission
