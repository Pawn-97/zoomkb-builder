# Call Queue Agent

## Summary

Similar to shared line group members, call queue agents can also use the *85 star code to opt in or out of their queue via IP phone. The same audible confirmation prompt applies. Designers should treat this role as equivalent in experience to the shared line group member for the star-code interaction.

## Key points

- Uses the same *85 dialing mechanism as shared line group members
- Receives the same audible confirmation prompt
