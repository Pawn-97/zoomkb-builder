# Contact Center Agent

## Summary

Receives enriched caller information when handling calls transferred from Zoom Phone. Agents see consumer identity, transfer source, IVR selections, and conversation summaries in incoming notifications, and can review the same information during active engagements and in completed records. This progressive disclosure supports workload management -- agents can preview before accepting and retrieve details later if they missed them.

## Key points

- Sees caller context in three views: incoming notification (before accepting), active engagement (during call), and completed record (after call)
- Enriched data reduces need for callers to repeat information
- Can review information post-acceptance if they didn't have time to review before accepting or if auto-accept is enabled
