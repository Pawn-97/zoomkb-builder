# End User (Zoom Client User)

## Summary

The only role that can validate ZPLS configuration on the local client. While admins configure the infrastructure, end users verify it works on their device. This validation step is currently buried in Statistics > Phone tab, which is not an intuitive location for non-technical users.

## Key points

- Validates ZPLS configuration on local Zoom client
- Accesses Settings > Statistics > Phone tab to check status
- Status indicators: Configured (setup complete) and Registered (connected)
