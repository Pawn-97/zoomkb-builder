# End User

## Summary

Zoom Phone users who receive a Zoom Phone Mobile license. After an admin issues their eSIM, the user receives an activation email with a QR code to install on their device. Users make and receive business calls via their phone's native dialer, with advanced Zoom Phone features available through the Zoom Workplace mobile app.

## Key points

- Receives activation email with QR code after admin assigns license
- Uses native dialer for business calls on mobile device
- Accesses advanced features (presence, voicemail, recording) via Zoom Workplace app
- SMS/MMS through device's native messaging app; Zoom app is read-only for messages
