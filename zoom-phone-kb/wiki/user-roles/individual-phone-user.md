# Individual Phone User

## Summary

Individual users can enable or disable call summary templates for their own account via Settings > Phone, unless locked by an admin at a higher level. The UX must clearly indicate when settings are locked (grayed out) and guide users to contact their admin for changes.

## Key points

- Can configure personal setting unless locked at group or account level
- Accesses setting via Settings > Phone > Settings > Others
- Does not require admin privileges to change their own setting
- Sees grayed-out toggle with explanation when locked by admin
