# Phone User (Call Queue Member)

## Summary

Phone users select their outbound caller ID from available numbers, but may encounter situations where expected numbers are absent from the dropdown. The UX must help users understand why a number is unavailable (not a call queue member, not assigned to their account) and provide a path to request access from an admin.

## Key points

- Selects outbound caller ID from a dropdown of available numbers
- Can only see numbers from their account, company main number, or their call queues
- May not understand why certain call queue numbers are absent from the dropdown
- Must contact admin if they need access to a call queue number
