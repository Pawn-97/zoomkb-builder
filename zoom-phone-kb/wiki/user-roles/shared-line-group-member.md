# Shared Line Group Member

## Summary

Users who belong to a shared line group and need to control whether they receive calls from that group. These users interact via IP phone star codes rather than a visual interface, so audible feedback is their sole confirmation mechanism. A user can only belong to one shared line group at a time, and cannot simultaneously be in a call delegation setup.

## Key points

- Must have a Zoom Phone license and a supported desk phone
- Can only belong to one shared line group—mutually exclusive with call delegation
- No visual UI; relies entirely on dial tones and voice prompts for feedback
