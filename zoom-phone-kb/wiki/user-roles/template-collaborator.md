# Template Collaborator

## Summary

A user explicitly added to a template's restricted collaboration list who can edit that template. This role enables distributed template management without granting full admin access, useful for subject matter experts who need to refine templates for their domain.

## Key points

- Has edit access to specific templates when added as a collaborator
- Only relevant when collaboration mode is set to 'Restricted'
- Managed in the Settings panel under Collaboration
