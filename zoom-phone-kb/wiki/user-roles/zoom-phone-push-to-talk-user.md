# Zoom Phone Push-to-Talk User

## Summary

End users who participate in push-to-talk channels and can trigger AI summaries. They interact with the AI Companion icon during calls and access summaries afterward in call history. The experience is mobile-first, available on Android and iOS.

## Key points

- Uses push-to-talk on Zoom Phone mobile app
- Can trigger AI summaries during or after push-to-talk calls
- Can select audio source (Bluetooth, Speaker, Mute, Headphones) before triggering summary
- Can edit and email generated summaries
