# Zoom Phone User (Allow List Member)

## Summary

Zoom Phone users who are added to the SMS Allow List gain the ability to view incoming SMS replies from customers. This role is distinct from the admin who configures the system—the UX must make clear that being on the Allow List is what grants visibility to customer SMS responses, not general Zoom Phone user status.

## Key points

- Must be explicitly added to the Allow List to view incoming SMS replies
- At least one user must be on the Allow List for the feature to function
- Distinct from general Zoom Phone user role; SMS reply visibility is opt-in
- No admin privileges required for this role
