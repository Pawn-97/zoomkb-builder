# Zoom Phone User (Transfer Initiator)

## Summary

When transferring a call to Contact Center, the Zoom Phone user's identity (name, extension, DID) and conversation summary are shared with the receiving agent based on admin configuration. Designers should consider whether users need visibility into what information is being shared about them and their conversation.

## Key points

- Caller name and caller ID are always shared on transfer
- Conversation summary sharing is configurable by admin
- Transfers can originate from individual users, queues, auto receptionists, or shared line groups
