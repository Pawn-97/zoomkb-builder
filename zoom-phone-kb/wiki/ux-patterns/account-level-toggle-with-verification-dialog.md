# Account-Level Toggle with Verification Dialog

## Summary

A simple binary toggle with a confirmation dialog, representing the most basic form of the Zoom Phone settings pattern. Unlike settings with hierarchical lock support, this toggle has no lock icon, no grayed-out states, and no group/site/user variants — making it the simplest setting configuration UX in the system.

## Key points

- Single toggle in Account Settings > Zoom Phone > General
- Verification dialog confirms the change (Enable/Disable)
- No lock icon or subordinate configuration levels
- Represents the baseline pattern for account-scoped settings
