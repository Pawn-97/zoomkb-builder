# Action Bar with Primary and Destructive Buttons

## Summary

The user detail page presents a set of action buttons at the top (Issue new eSIM, Deactivate, Enable), clearly separating primary actions from destructive ones. The irreversible Deactivate action is noted explicitly.

## Key points

- Top action bar with multiple action buttons
- Deactivate noted as irreversible
- Enable/Deactivate toggle pattern based on current state
