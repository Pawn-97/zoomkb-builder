# Admin Portal Sync Action Pattern

## Summary

The sync action is accessed through a consistent drill-down path (Phone System Management > Company Info > Site > Settings > Zoom Node > Local Survivability > Manage) with the Sync button contextualized by scope. For individual nodes the Sync appears in the Status section; for node groups it appears in the Site section. This contextual placement helps admins understand what scope they are syncing.

## Key points

- Consistent drill-down navigation to Local Survivability settings
- Context-aware Sync button placement (Status section for nodes, Site section for groups)
- Nested management via 'Manage' link under Zoom Node section
