# Admin-Supervised Self-Service Recovery

## Summary

Users can self-serve QR code regeneration for planned device upgrades, but lost-device scenarios require admin intervention (New eSIM). This graduated escalation pattern balances user autonomy with security controls.

## Key points

- Self-service: 'View QR Code' / regenerate QR for planned upgrades
- Admin-only: 'New eSIM' for lost/stolen devices
- Clear distinction in admin UI between the two actions
