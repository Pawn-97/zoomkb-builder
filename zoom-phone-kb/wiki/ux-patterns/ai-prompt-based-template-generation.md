# AI Prompt-Based Template Generation

## Summary

Admins describe desired call summary format in natural language, and AI generates a structured template from the prompt. This is a text-to-configuration pattern where prompt quality determines output quality. The UX should include prompt guidance, examples, and robust review/edit capabilities since AI output may need refinement.

## Key points

- Natural language prompt defines template format, sections, highlights, and terminology
- AI generates a complete template that can be reviewed and adjusted before saving
- Post-generation editing allows correcting AI misinterpretations
