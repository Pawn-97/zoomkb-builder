# Alert Configuration Wizard with Conditional Sections

## Summary

A multi-section form where the Rule selection dynamically changes which Target options and parameters appear. This is a classic conditional form pattern where field visibility depends on a prior selection. The form is organized into labeled sections (Rule, Notification, Active Status) with a final Save action, providing clear structure despite the complexity.

## Key points

- Three major sections: Rule, Notification, Active Status
- Rule dropdown selection drives conditional target options and parameter fields
- Section labeling provides scannable structure for a complex form
- Single Save button at the bottom commits all settings at once
