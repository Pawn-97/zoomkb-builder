# Audible Confirmation Prompt for State Changes

## Summary

After dialing the star code, the system plays a voice prompt confirming the user's new call participation state. Since there is no screen on many desk phones, this audible feedback is the sole confirmation mechanism. The prompt must be unambiguous ('opted in' vs. 'opted out') and play immediately after the digits are sent.

## Key points

- Voice prompt plays immediately after star code digits are sent
- Two distinct messages: 'You are now opted out' and 'You are now opted in'
- Serves as the only state-change confirmation for users without a screen interface
