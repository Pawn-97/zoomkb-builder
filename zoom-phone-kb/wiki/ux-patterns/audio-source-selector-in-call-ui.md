# Audio Source Selector in Call UI

## Summary

A selector in the top-left corner of the call screen allowing users to choose between Bluetooth, Speaker, Mute, and Headphones. This follows a standard media control pattern and must be easily accessible during an active push-to-talk session.

## Key points

- Located in top-left corner of call screen
- Options include Bluetooth, Speaker, Mute, and Headphones
- Optional step — user can proceed without changing audio source
