# Auto-Created Device Endpoint in Call Handling

## Summary

Assigning a Zoom Phone Mobile license automatically creates a mobile device endpoint in the user's call handling settings. This ensures the mobile device appears as a destination in call routing and handling configurations without additional admin setup.

## Key points

- Mobile endpoint auto-created on license assignment
- Integrates into existing call handling and routing UX
- Reduces manual device registration steps
