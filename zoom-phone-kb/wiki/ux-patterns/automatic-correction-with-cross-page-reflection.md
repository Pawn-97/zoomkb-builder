# Automatic Correction with Cross-Page Reflection

## Summary

When the system corrects a device model, the updated information propagates to all web portal pages automatically. Designers should ensure state consistency across views without requiring manual refresh, providing a seamless admin experience.

## Key points

- Corrections reflected instantly across all portal pages
- No manual refresh or reload required by the admin
- Affects device detail pages, lists, and any other view showing device models
