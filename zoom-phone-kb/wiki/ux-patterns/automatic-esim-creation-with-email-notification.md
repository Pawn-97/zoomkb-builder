# Automatic eSIM Creation with Email Notification

## Summary

When a Zoom Phone Mobile license is first assigned to a user, the system automatically creates an eSIM, generates a QR code, and sends a welcome email to the user. This zero-touch provisioning pattern removes manual admin steps for initial setup.

## Key points

- eSIM auto-created on license assignment
- QR code delivered via email for user self-activation
- Reduces admin burden for initial provisioning
