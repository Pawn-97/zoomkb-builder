# Automatic QR Code Email on Provisioning

## Summary

Creating an eSIM package or issuing a new eSIM automatically triggers a welcome email containing a QR code to the user. This zero-touch pattern delegates device setup to the end user while keeping the admin informed that provisioning is complete.

## Key points

- QR code delivered via email automatically
- No admin intervention needed after creation
- User self-service activation reduces support burden
