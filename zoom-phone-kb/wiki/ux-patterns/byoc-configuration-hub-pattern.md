# BYOC Configuration Hub Pattern

## Summary

The Number Management > BYOC Configuration area serves as a hub for all SBC-related management, including Session Border Controllers and Route Groups. Each section has its own Manage link, creating a consistent pattern of drill-down configuration panels. The separation between SBC configuration and Route Group management is logical but could benefit from a setup wizard linking the two.

## Key points

- BYOC Configuration as centralized hub for SBC management
- Separate Manage links for Session Border Controllers and Route Groups
- Survivability configuration embedded within standard SBC and route group interfaces
