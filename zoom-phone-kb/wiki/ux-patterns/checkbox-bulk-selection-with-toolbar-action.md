# Checkbox Bulk Selection with Toolbar Action

## Summary

Bulk operations use checkbox selection on each row, with a select-all checkbox in the column header. A contextual toolbar button ('Move numbers between master/sub account') appears or becomes active when selections are made.

## Key points

- Per-row checkboxes plus a select-all checkbox in the column header
- Toolbar action button activates on selection
- Pop-up window handles additional steps like address verification
