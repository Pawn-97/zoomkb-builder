# Cross-System Configuration (ZVA + Zoom Phone)

## Summary

Setting up SMS requires configuration in two separate admin interfaces: Zoom Virtual Agent (voice agent settings) and Zoom Phone (auto receptionist settings). This split-configuration pattern increases cognitive load and the risk of partial setup. Designers should consider a unified setup wizard that guides admins through both sides or at minimum cross-links between the two configuration surfaces.

## Key points

- Configuration split between ZVA voice agent builder and Phone System Management
- Each side has distinct prerequisites and steps
- No mention of a unified or wizard-based setup flow
- Admins must understand the relationship between the two systems to complete setup
- Cross-linking between the two configuration surfaces would reduce confusion
