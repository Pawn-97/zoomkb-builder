# Dropdown Distribution Selector

## Summary

The call distribution method is selected via a dropdown control. Changing from the default (e.g., Simultaneous) to Group Rotating reveals additional dependent controls: group member size and ringing duration. This progressive disclosure pattern keeps the UI clean until the relevant option is chosen.

## Key points

- Dropdown toggle between distribution modes
- Progressive disclosure: numeric inputs appear only when Group Rotating is selected
- Confirm button required to apply the distribution change within the section
