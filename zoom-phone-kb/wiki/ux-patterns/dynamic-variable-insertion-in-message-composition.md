# Dynamic Variable Insertion in Message Composition

## Summary

The SMS content field supports dynamic variables (e.g., recipient phone number, custom text) inserted via the same forward-slash or Insert mechanism. This allows admins to compose template messages that personalize at runtime. The UX must provide a clear variable picker with descriptive labels so non-technical users understand what each variable represents.

## Key points

- Variables inserted into SMS content using forward-slash or Insert button
- Recipient phone number is configured as a variable, not a static value
- Dynamic text variables enable personalized messaging at scale
- Variable names should be descriptive to guide non-technical admins
