# Ellipsis Menu for Row-Level Actions

## Summary

Each user row in the eSIM Management list uses an ellipsis menu to surface secondary actions (e.g., View). This keeps the list view clean while making contextual actions discoverable.

## Key points

- Ellipsis icon per table row
- Clicking reveals contextual menu options
- Used for drill-down navigation (View) rather than inline editing
