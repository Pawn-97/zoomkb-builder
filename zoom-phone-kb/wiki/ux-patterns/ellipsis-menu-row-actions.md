# Ellipsis Menu Row Actions

## Summary

Individual actions on phone number rows are accessed via an ellipsis (three-dot) menu, which opens a contextual action (Move to account). This pattern keeps the list view clean while surfacing per-item actions.

## Key points

- Ellipsis icon per table row
- Clicking opens a context menu with 'Move to account' option
- Triggers a pop-up modal/window for the action flow
