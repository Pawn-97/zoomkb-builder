# Field Mapping Between App Data Models

## Summary

After selecting a trigger, Zapier pulls available Zoom Phone fields (caller name, number, direction, timestamp, etc.) for mapping into subsequent action steps. Users can insert these dynamic values into third-party app fields, creating data pipelines without code.

## Key points

- Dynamic field discovery: Zapier surfaces available Zoom Phone fields automatically
- Users insert fields into downstream app templates (e.g., Gmail subject line)
- Field labels should be self-explanatory for non-technical users
