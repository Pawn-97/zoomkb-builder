# Floating Phone Window (Smart Embed)

## Summary

Zoom Phone opens in a floating window within LTI Pro, sharing the same interaction model as Zoom Phone Smart Embed. This pattern allows telephony functionality to live alongside LMS course content without leaving the page. The floating window handles calls, SMS, voicemail, and call history in a compact persistent panel.

## Key points

- Floating panel that overlays LMS course content
- Contains Contacts, Phone, and SMS tabs
- Shares design and behavior with Zoom Phone Smart Embed
