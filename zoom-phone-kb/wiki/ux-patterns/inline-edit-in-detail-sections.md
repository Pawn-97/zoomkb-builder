# Inline Edit in Detail Sections

## Summary

Within the General Info section of the user detail page, editable fields (user name, phone number) feature an Edit button that enables inline editing rather than navigating to a separate form.

## Key points

- Edit buttons inline next to field labels
- Limited to specific fields (user name, phone number)
- Read-only fields (Package, Status) have no Edit button
