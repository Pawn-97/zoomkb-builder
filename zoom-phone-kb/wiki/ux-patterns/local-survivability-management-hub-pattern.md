# Local Survivability Management Hub Pattern

## Summary

The Local Survivability Manage screen serves as a hub with multiple tabs (including Node groups) for configuring different aspects of ZPLS. This tab-based organization groups related survivability configuration together while keeping each concern in its own tab. The 'New' button and '...' (More) menu on rows provide consistent CRUD interaction patterns.

## Key points

- Local Survivability > Manage screen as central hub for ZPLS configuration
- Tab-based organization (Node groups tab alongside other ZPLS config tabs)
- New button for creation, '...' (More) menu for row-level edit actions
- Modal windows for edit operations
