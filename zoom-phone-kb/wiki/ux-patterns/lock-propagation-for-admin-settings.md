# Lock Propagation for Admin Settings

## Summary

A lock icon toggle allows admins to prevent downstream users from overriding account-level settings. The lock visual is a common admin UX pattern that communicates policy enforcement versus personal preference, and should be consistently applied across all settings that support hierarchical governance.

## Key points

- Lock icon appears next to settings that can be enforced at the account level
- Locking prevents all users in the account from changing the setting
- Requires confirmation dialog to prevent accidental locking
