# Manage Connections and Added Apps Pattern

## Summary

Connections are managed across two surfaces: Zapier's Manage Connections page lists all connected accounts, and Zoom's App Marketplace shows Added Apps. This dual-management pattern is common for marketplace integrations but requires users to understand both surfaces.

## Key points

- Zapier: Manage Connections page for account-level disconnection
- Zoom: App Marketplace > Manage > Added Apps for complete removal
- Design consideration: provide clear cross-platform removal instructions
