# Manage Pattern for Site-Level Feature Configuration

## Summary

The consistent pattern of navigating to a Site, opening Settings, and finding Zoom Node sub-features with a 'Manage' link is repeated across all call forwarding interactions. This establishes the Site Settings page as a hub for all ZPLS-related features, with Manage links acting as drill-down entry points to feature-specific configuration panels.

## Key points

- Site > Settings > Zoom Node serves as hub for ZPLS features
- 'Manage' link is the consistent entry point for detailed configuration
- Consistent across configure and enable workflows
