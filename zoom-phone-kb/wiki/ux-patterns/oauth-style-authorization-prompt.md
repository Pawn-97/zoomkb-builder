# OAuth-Style Authorization Prompt

## Summary

The first-time Zoom Phone authorization in LTI Pro uses an OAuth-style redirect: the user clicks Connect, is redirected to log in, and must click Allow to grant permissions. A success message confirms the connection. This familiar delegation pattern should follow platform OAuth conventions for trust and clarity.

## Key points

- Connect button triggers redirect to Zoom login
- Allow/Deny permission prompt after login
- Success confirmation message on completion
