# Pop-Up Modal for Create and Confirmation Flows

## Summary

Creating eSIM packages, issuing new eSIMs, and managing roaming dates all use pop-up modal windows. This pattern keeps the admin anchored in context while performing focused tasks.

## Key points

- Create user flow: pop-up with dropdown fields (Username, Package, Number)
- Confirmation modals for destructive actions (deactivate, delete roaming)
- Roaming date edits: pop-up with date picker fields
