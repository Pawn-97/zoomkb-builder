# QR Code with Manual Fallback

## Summary

Primary activation path uses QR code scanning for simplicity, with a secondary manual code entry path for devices or situations where scanning is not possible. This ensures all users can complete activation regardless of hardware limitations or environment.

## Key points

- Primary path: QR code scan (camera or SIM settings) -- fast and error-free
- Fallback path: manual entry of SM-DP+ address and activation/network provider code
- Both paths lead to the same device-native activation flow
