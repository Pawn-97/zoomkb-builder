# Related Features Dropdown Navigation

## Summary

Secondary or related features (like Business Address & Documents) are accessed through a dropdown menu on the Phone Numbers page. This pattern surfaces non-primary actions that are contextually related to the current page. However, when these features are prerequisites for the primary action, the dropdown may not provide enough visibility.

## Key points

- Dropdown labeled 'Related Features' on the Phone Numbers page
- Contains links to Business Address & Documents and potentially other features
- Discovery of prerequisite features depends on admin recognizing the relationship
