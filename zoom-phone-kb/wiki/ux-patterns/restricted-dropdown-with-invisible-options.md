# Restricted Dropdown with Invisible Options

## Summary

The outbound caller ID dropdown only shows numbers the user is permitted to use, silently hiding unavailable options without explanation. This creates a discoverability problem where users may not realize certain numbers exist or understand why they cannot access them. A more transparent pattern would show unavailable options as disabled with explanatory tooltips.

## Key points

- Dropdown shows only permitted numbers, not all possible numbers
- No indication that additional numbers exist but are unavailable
- Users must infer or ask admin why a number is missing
- 100-item cap adds another layer of potential invisibility
