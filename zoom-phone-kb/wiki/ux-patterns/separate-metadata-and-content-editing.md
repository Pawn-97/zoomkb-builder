# Separate Metadata and Content Editing

## Summary

Template metadata (icon, name, description, user assignment) is edited via a pencil icon pop-up, while template content (sections, formatting, terminology) is edited directly on the template view. This separation helps admins understand which changes affect template appearance/availability vs. the actual output format.

## Key points

- Pencil icon triggers metadata editing popup (icon, name, description, user assignment, delete)
- Content editing is done inline on the template page (length, sections, terminology)
- Settings icon opens a panel for collaboration configuration
