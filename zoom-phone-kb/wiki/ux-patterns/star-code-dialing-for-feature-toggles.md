# Star-Code Dialing for Feature Toggles

## Summary

A telephony interaction pattern where users dial a numeric code on a physical keypad to toggle a system feature, similar to legacy PBX star codes. The pattern uses a prefix (*85) followed by a target identifier (extension number). This is a headless, non-visual interaction that relies entirely on DTMF tones and audible feedback.

## Key points

- Two-part dialing: feature prefix (*85) + target identifier (extension number)
- No visual interface required—entirely DTMF and audio-based
- Follows established telephony star-code conventions for familiarity
