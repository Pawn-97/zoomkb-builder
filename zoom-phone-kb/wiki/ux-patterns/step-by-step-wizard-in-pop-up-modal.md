# Step-by-Step Wizard in Pop-up Modal

## Summary

The phone number transfer uses a multi-step wizard pattern within a pop-up window. Step 1 collects configuration (account, product, subscription, etc.), and Step 2 handles address verification. The Next and Move buttons guide the user through sequential steps.

## Key points

- Multi-step flow in a modal window
- Next button advances to confirmation step
- Clear primary action (Move) completes the flow
- Dropdown-based field selection throughout
