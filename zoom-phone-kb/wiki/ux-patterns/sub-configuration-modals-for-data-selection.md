# Sub-Configuration Modals for Data Selection

## Summary

Complex data options (CRM ticket info, Variables, Address book custom fields) use sub-configuration flows via 'Manage Variable', '+ Add variable(s)', and '+ Add custom field(s)' buttons. This pattern keeps the main configuration page clean while allowing deep customization through drill-down modals.

## Key points

- Main configuration shows high-level data options as checkboxes
- Options with sub-selections trigger separate modal/pane for field selection
- Each sub-configuration saves independently before the main Save
