# Tab-Based Queue Configuration

## Summary

Call queue settings are organized into tabs (e.g., Profile tab). Within a tab, related settings are grouped into logical sections (e.g., Business Hours). This pattern organizes complex configuration into manageable, scannable groups.

## Key points

- Profile tab contains the Business Hours section with call distribution settings
- Section headers visually group related controls
- Incremental save pattern: Confirm individual changes within a section before saving the whole queue
