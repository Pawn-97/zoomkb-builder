# Template Selection with Named Variants

## Summary

Predefined templates are offered as named, purpose-specific options (General, One-on-one, Project update, Discussion, Customer success), each with a short description of its focus. This pattern helps admins quickly identify which template fits their use case without needing to preview each one.

## Key points

- Five named templates with short purpose descriptions
- Each targets a specific call type (general, 1:1 meetings, projects, discussions, customer interactions)
- Reduces cognitive load by offering curated starting points
