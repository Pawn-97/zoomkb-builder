# Terminology Gap Between Feature Name and Control Label

## Summary

The feature is described as 'Transferring call data to Contact Center,' but the actual UI control is labeled 'Conversation summary.' This terminology mismatch could cause confusion for admins looking to enable the feature. Designers should align the setting's visible label with the terminology used in documentation and navigation.

## Key points

- Feature name: 'Transferring call data to Contact Center'
- Control label: 'Conversation summary'
- Mismatch may cause admins to overlook the setting
- Consistent terminology across documentation and UI reduces confusion
