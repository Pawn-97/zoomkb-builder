# Test-Before-Publish Workflow

## Summary

Templates must be tested before being published to end users, creating a draft-to-live lifecycle. This pattern is critical for AI-generated content where output quality is not deterministic and admins need to verify that templates produce useful summaries before users rely on them.

## Key points

- Templates can be created and edited in a draft state
- Testing verifies template output before release
- Publishing makes the template available to assigned users
