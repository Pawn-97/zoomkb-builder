# Toggle for Enable/Disable (Active Status)

## Summary

Each alert has an Active Status toggle that enables or disables the alert without deleting it. New alerts default to active. This soft-disable pattern allows admins to temporarily silence an alert while preserving its configuration for later reactivation.

## Key points

- Single toggle control for enable/disable state
- New alerts default to active
- Allows pausing alerts without losing configuration
