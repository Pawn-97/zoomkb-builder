# Tool Insertion via Search and Forward-Slash Shortcut

## Summary

Within the voice agent instruction builder, tools and variables are inserted either by clicking Insert and searching, or by typing a forward-slash (/) to invoke an inline picker. This dual-path insertion pattern (mouse + keyboard shortcut) provides efficiency for power users while remaining discoverable for new users. The forward-slash pattern is common in modern editing interfaces but may need onboarding cues.

## Key points

- Forward-slash (/) keyboard shortcut for quick tool/variable insertion
- Insert button provides a click-based alternative for discoverability
- Search-as-you-type within the insertion interface
- Pattern likely extends to other tools beyond SMS
