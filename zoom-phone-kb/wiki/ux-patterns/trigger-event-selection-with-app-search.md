# Trigger-Event Selection with App Search

## Summary

In the Zap editor, users search for Zoom Phone, select it, and then choose from a list of predefined trigger events. This search-and-select pattern simplifies discovery of the right integration among thousands of apps, and the event list provides a curated set of meaningful trigger points.

## Key points

- Search bar to find Zoom Phone among Zapier's app catalog
- Event list with human-readable event names
- Covers the call lifecycle: connect, end, complete, recording done, voicemail received
