# Two-Path Destructive Action (Inline + Page-Level Delete)

## Summary

Alerts can be deleted via two paths: an inline ellipsis menu on the list view, or a Delete button at the bottom of the detail page. Both paths show a confirmation dialog. This dual-path pattern gives users flexibility but requires consistent confirmation behavior across both entry points.

## Key points

- Inline ellipsis menu on alert list row for quick deletion
- Page-level Delete button at the bottom of the alert detail view
- Confirmation dialog required before deletion in both paths
