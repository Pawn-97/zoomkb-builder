# Verification Dialog on Toggle

## Summary

When toggling delegation on or off, a verification dialog appears requiring explicit confirmation. This pattern prevents accidental changes to settings that have broad downstream impact, particularly important for Account and Group level changes that affect many users.

## Key points

- Appears after toggling delegation at any level
- Requires clicking Enable or Disable to confirm the change
- Acts as a safety net for high-impact settings changes
