# Verification Status Indicator

## Summary

The Business Address & Documents page displays a Verification Status field that reads 'Verified' when the review is complete. This status indicator is a critical checkpoint that admins must confirm before proceeding to purchase numbers. Designers should consider making this status visible from the number selection flow itself.

## Key points

- Verification Status shown on the Business Address & Documents page
- Status must be 'Verified' before numbers can be purchased
- Currently located one navigation level away from the number selection flow
